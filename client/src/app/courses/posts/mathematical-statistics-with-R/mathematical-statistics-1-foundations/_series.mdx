## Preface

This is a book about mathematics and statistics. It is written for people who already know how to programme in R but want to understand — deeply and rigorously — the statistical methods they use.

We live in an era where large language models can generate statistical code on demand. This makes it more important, not less, to understand the mathematics behind statistical methods. If you cannot evaluate whether an analysis is correct, you cannot trust any tool that produces one — whether that tool is software, a colleague, or an AI.

This book takes the mathematics seriously. We state definitions precisely, prove theorems fully, and derive every formula from first principles. But we also believe that mathematical rigour and clear communication are not opposites. Every concept is introduced in plain language before it is formalised. We use both registers: the precise language of mathematics and the everyday language of explanation.

We then go further. After each derivation, we implement the method in R — not by calling a built-in function, but by building it from scratch. You will code your own estimators, your own tests, your own regression algorithms. Only after you understand what a function does will we show you the standard R implementation. Visualisations in ggplot2 and interactive animations in JavaScript reinforce each concept from yet another angle.

The pedagogical method is repetition through different modes of understanding:

1. **Prose and intuition** — what is this concept, why does it matter, when would you use it?
2. **Formal mathematics** — definitions, theorems, proofs, derivations in LaTeX
3. **Implementation from scratch** — build it yourself in R before using built-in functions
4. **Visualisation** — ggplot2 plots and JavaScript canvas animations
5. **Exercises** — proofs, computations, interpretations, and error-spotting

This is Book I of three. It covers probability, distributions, estimation, hypothesis testing, linear models, generalised linear models, and an introduction to Bayesian statistics. Book II covers advanced methods. Book III covers research-level topics.

### Prerequisites

- Working knowledge of R (data structures, functions, ggplot2)
- Basic calculus (derivatives, integrals) — we review key results as needed
- Comfort with summation and product notation

All other mathematics — linear algebra, set theory, combinatorics, the Gamma function — is introduced when the statistics demands it.
