# Chapter 18 -- Analysis of Variance

## Part 2: Factorial Designs and ANCOVA

One-way ANOVA tests whether a single factor affects the response. In practice, experiments often involve **two or more factors simultaneously**. The power of factorial designs lies in their ability to estimate not only the effect of each factor, but also their **interaction** -- the extent to which the effect of one factor depends on the level of another. ANCOVA extends the framework to include continuous covariates alongside categorical factors. All of these are simply different flavours of the general linear model.

---

## 18.13 Two-Way ANOVA: The Model

### 18.13.1 Setup

With factors $A$ (with $a$ levels) and $B$ (with $b$ levels), each combination observed $n$ times, the model is:

$$Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \varepsilon_{ijk}$$

where:
- $\mu$ is the overall mean.
- $\alpha_i$ is the main effect of factor $A$ at level $i$ ($\sum_i \alpha_i = 0$).
- $\beta_j$ is the main effect of factor $B$ at level $j$ ($\sum_j \beta_j = 0$).
- $(\alpha\beta)_{ij}$ is the interaction ($\sum_i (\alpha\beta)_{ij} = 0$ for each $j$, and vice versa).
- $\varepsilon_{ijk} \stackrel{\text{iid}}{\sim} N(0, \sigma^2)$.

### 18.13.2 The ANOVA Decomposition

The total sum of squares decomposes as:

$$\text{SST} = \text{SS}_A + \text{SS}_B + \text{SS}_{AB} + \text{SSE}$$

with degrees of freedom:

$$abn - 1 = (a-1) + (b-1) + (a-1)(b-1) + ab(n-1)$$

Each sum of squares, divided by its df, gives a mean square. The $F$-tests are:

| Effect | $F$-statistic | $H_0$ |
|:-------|:-------------|:------|
| $A$ | $\text{MS}_A / \text{MSE}$ | All $\alpha_i = 0$ |
| $B$ | $\text{MS}_B / \text{MSE}$ | All $\beta_j = 0$ |
| $A \times B$ | $\text{MS}_{AB} / \text{MSE}$ | All $(\alpha\beta)_{ij} = 0$ |

### 18.13.3 As Regression

The two-way ANOVA model with interaction is:

```
Y ~ A + B + A:B
```

which R shorthand writes as `Y ~ A * B`. The design matrix contains dummy columns for each level of $A$, each level of $B$, and their pairwise products.

## 18.15 Type I, II, and III Sums of Squares

### 18.15.1 The Issue

When the design is **unbalanced** (unequal cell sizes), the sums of squares for the different effects are not orthogonal, and the order in which terms are added to the model matters. R's `anova()` computes **Type I** (sequential) sums of squares by default.

### 18.15.2 The Three Types

- **Type I** (sequential): each effect is tested after adjusting for all preceding effects. The results depend on the order of terms.
- **Type II**: each main effect is tested after adjusting for the other main effects, but **not** for the interaction.
- **Type III**: each effect is tested after adjusting for all other effects **including** the interaction. This requires properly centred contrasts.

### 18.15.3 Demonstrating the Difference


``` r
# Type I: order matters for unbalanced designs
anova(lm(len ~ supp + dose + supp:dose, data = tg))
anova(lm(len ~ dose + supp + dose:supp, data = tg))
```

### 18.15.4 Practical Advice

- For **balanced designs**: all three types give the same results.
- For **unbalanced designs** with no interaction: Type II is generally preferred (more powerful).
- For **unbalanced designs** with a significant interaction: interpret the interaction directly rather than relying on main effect tests.

## 18.16 ANCOVA: Continuous + Categorical

### 18.16.1 The Idea

**Analysis of Covariance** (ANCOVA) combines ANOVA with regression. The model includes both categorical predictors (factors) and continuous predictors (covariates):

$$Y_{ij} = \mu + \alpha_j + \beta x_{ij} + \varepsilon_{ij}$$

The covariate $x$ is included to reduce residual variance and/or to adjust for a confounding variable. ANCOVA answers: "Do the group means differ **after adjusting for** the covariate?"

### 18.16.2 Geometric Interpretation

ANCOVA fits **parallel regression lines** (one per group). The lines share a common slope $\beta$ but have different intercepts $\mu + \alpha_j$. If the interaction `factor:covariate` is included, the lines have different slopes -- this is the "heterogeneous slopes" model.

### 18.16.3 Assumptions

In addition to the standard regression assumptions:
1. The relationship between $Y$ and the covariate is linear within each group.
2. The slopes are equal across groups (homogeneity of slopes) -- unless the interaction is explicitly modelled.

## 18.22 The Hierarchy Principle

When an interaction is significant, the main effects involved in that interaction should **always** be retained in the model, even if their individual $p$-values are large. This is the **hierarchy (or marginality) principle**: the interpretation of an interaction $A \times B$ is meaningful only if both $A$ and $B$ are in the model.


``` r
# Wrong: dropping main effect but keeping interaction
fit_wrong <- lm(len ~ supp:dose, data = tg)

# Correct: keep main effects when interaction is present
fit_right <- lm(len ~ supp * dose, data = tg)

c(
  "R2 without main effects" = summary(fit_wrong)$r.squared,
  "R2 with main effects" = summary(fit_right)$r.squared
)
```

## 18.24 Connecting ANOVA to the Broader Framework

| What you call it | What it really is |
|:----------------|:-----------------|
| One-way ANOVA | `lm(Y ~ factor)` |
| Two-way ANOVA | `lm(Y ~ A + B)` |
| Factorial ANOVA | `lm(Y ~ A * B)` |
| ANCOVA | `lm(Y ~ factor + covariate)` |
| Repeated measures | Mixed-effects model (beyond this chapter) |

Every entry in this table is a special case of the general linear model. The $F$-tests are all instances of comparing nested models. Understanding this unity is the single most important insight from this chapter.

---

## Communicating to Stakeholders

When presenting factorial ANOVA and ANCOVA results:

- **Lead with the interaction plot.** It communicates more than any table of numbers. If the lines are parallel, say "the effect of dose is the same for both supplements." If they cross or diverge, say "the effect of dose depends on which supplement was used."
- **Beware of marginal means when interactions are present.** If you report "the mean for OJ is 20.7 and the mean for VC is 17.0," you are averaging over doses -- which is misleading when the supplement effect depends on dose.
- **ANCOVA adjusts for confounders.** "After adjusting for BMI, blood pressure still increases with age" is a more precise statement than "blood pressure increases with age."
- **Report both the ANOVA table and the model coefficients.** The ANOVA table gives the overall tests; the coefficients give the effect sizes and directions.

## Quick Reference

| Quantity | Formula |
|:---------|:--------|
| **Two-way model** | $Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \varepsilon_{ijk}$ |
| **ANCOVA model** | $Y_{ij} = \mu + \alpha_j + \beta x_{ij} + \varepsilon_{ij}$ |
| **Interaction $F$** | $\text{MS}_{AB} / \text{MSE}$ |
| **Type I SS** | Sequential -- depends on term order |
| **Type II SS** | Each main effect adjusted for other main effects |
| **Type III SS** | Each effect adjusted for all others including interactions |
| **Hierarchy principle** | Keep main effects when their interaction is in the model |
