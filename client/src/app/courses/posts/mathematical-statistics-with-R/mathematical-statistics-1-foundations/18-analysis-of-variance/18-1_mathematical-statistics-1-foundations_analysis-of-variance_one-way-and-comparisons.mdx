# Chapter 18 -- Analysis of Variance

## Part 1: One-Way ANOVA and Comparisons

Analysis of Variance (ANOVA) is among the most widely used procedures in experimental science. It is often taught as a standalone technique, but the deepest understanding comes from recognising that **ANOVA is regression with categorical predictors**. The $F$-test in ANOVA is exactly the $F$-test for comparing a full model against a reduced (intercept-only) model. In this part we derive the one-way ANOVA $F$-test as a nested model comparison, show that `aov()` and `anova(lm())` give identical results, and connect multiple comparisons to Chapter 12.

---

## 18.1 The One-Way Layout

### 18.1.1 Setup

We observe $n_{j}$ responses in each of $k$ groups (levels of a factor), for a total of $N = \sum_{j=1}^k n_j$ observations. The model is:

$$Y_{ij} = \mu_j + \varepsilon_{ij}, \quad i = 1, \ldots, n_j, \quad j = 1, \ldots, k$$

where $\varepsilon_{ij} \stackrel{\text{iid}}{\sim} N(0, \sigma^2)$.

Equivalently, using the "effects" parametrisation with overall mean $\mu$ and group effects $\alpha_j$:

$$Y_{ij} = \mu + \alpha_j + \varepsilon_{ij}, \quad \sum_{j=1}^k n_j \alpha_j = 0$$

### 18.1.2 The Hypothesis

$$H_0: \mu_1 = \mu_2 = \cdots = \mu_k \quad \text{(all group means are equal)}$$
$$H_1: \text{at least two means differ}$$

Under the effects parametrisation, $H_0: \alpha_1 = \alpha_2 = \cdots = \alpha_k = 0$.

---

## 18.2 ANOVA as Regression

### 18.2.1 Dummy Coding

A categorical variable with $k$ levels is encoded using $k - 1$ dummy (indicator) variables. If group $j = 1$ is the reference category, define:

$$D_{ij}^{(m)} = \begin{cases} 1 & \text{if observation } i \text{ belongs to group } m \\ 0 & \text{otherwise} \end{cases}, \quad m = 2, \ldots, k$$

The regression model becomes:

$$Y_i = \beta_0 + \beta_2 D_i^{(2)} + \beta_3 D_i^{(3)} + \cdots + \beta_k D_i^{(k)} + \varepsilon_i$$

where $\beta_0 = \mu_1$ (the reference group mean) and $\beta_m = \mu_m - \mu_1$ (the difference between group $m$ and the reference).

### 18.2.2 The Full and Reduced Models

- **Full model** (one mean per group): $Y_i = \beta_0 + \beta_2 D_i^{(2)} + \cdots + \beta_k D_i^{(k)} + \varepsilon_i$ -- this has $p = k$ parameters.
- **Reduced model** (one common mean): $Y_i = \beta_0 + \varepsilon_i$ -- this has $p_0 = 1$ parameter.

Testing $H_0: \beta_2 = \cdots = \beta_k = 0$ is equivalent to comparing these two nested models.

### 18.2.3 Deriving the $F$-test

The general $F$-statistic for comparing nested models is:

$$F = \frac{(\text{SSE}_{\text{reduced}} - \text{SSE}_{\text{full}}) / (p - p_0)}{\text{SSE}_{\text{full}} / (N - p)}$$

For the ANOVA problem:

- $\text{SSE}_{\text{reduced}} = \text{SST} = \sum_{j=1}^k \sum_{i=1}^{n_j}(Y_{ij} - \bar{Y}_{\cdot\cdot})^2$ (total SS).
- $\text{SSE}_{\text{full}} = \text{SSW} = \sum_{j=1}^k \sum_{i=1}^{n_j}(Y_{ij} - \bar{Y}_{\cdot j})^2$ (within-group SS).
- $\text{SST} - \text{SSW} = \text{SSB} = \sum_{j=1}^k n_j(\bar{Y}_{\cdot j} - \bar{Y}_{\cdot\cdot})^2$ (between-group SS).
- $p - p_0 = k - 1$, $N - p = N - k$.

Therefore:

$$\boxed{F = \frac{\text{SSB}/(k-1)}{\text{SSW}/(N-k)} = \frac{\text{MSB}}{\text{MSW}} \sim F_{k-1, N-k} \quad \text{under } H_0}$$

This is the one-way ANOVA $F$-test. Under $H_0$, the ratio of mean squares follows an $F$-distribution because both the numerator and denominator are independent $\chi^2$ random variables (divided by their degrees of freedom), by the properties of normal quadratic forms.

## 18.3 Building the ANOVA Table from Scratch


``` r
data_dir <- "../../../data"
penguins_raw <- data.table$fread(file.path(data_dir, "primary/penguins.csv"))
penguins <- penguins_raw[
  complete.cases(penguins_raw[, .(flipper_length_mm, species)])
]

# One-way ANOVA: flipper length by species
y <- penguins$flipper_length_mm
g <- penguins$species

groups <- unique(g)
k <- length(groups)
N <- length(y)
grand_mean <- mean(y)

# Group means and sizes
group_stats <- penguins[, .(
  mean = mean(flipper_length_mm),
  n = .N
), by = species]

# Sums of squares
SST <- sum((y - grand_mean)^2)
SSB <- sum(group_stats$n * (group_stats$mean - grand_mean)^2)
SSW <- SST - SSB

# Mean squares
df_between <- k - 1
df_within <- N - k
MSB <- SSB / df_between
MSW <- SSW / df_within

# F statistic and p-value
F_stat <- MSB / MSW
p_value <- pf(F_stat, df_between, df_within, lower.tail = FALSE)

anova_table <- data.table$data.table(
  Source = c("Between (species)", "Within (residual)", "Total"),
  df = c(df_between, df_within, N - 1),
  SS = c(SSB, SSW, SST),
  MS = c(MSB, MSW, NA),
  `F` = c(F_stat, NA, NA),
  `p-value` = c(p_value, NA, NA)
)

anova_table
```

### 18.4.3 Verifying Equivalence


``` r
# The F-statistics should be identical
c(
  "F from scratch" = F_stat,
  "F from aov" = summary(fit_aov)[[1]]["species", "F value"],
  "F from anova(lm)" = anova(fit_lm)["species", "F value"]
)
```

```
  F from scratch       F from aov F from anova(lm) 
        594.8016         594.8016         594.8016 
```

All three approaches produce the same $F$-statistic. The `aov()` function is simply a wrapper around `lm()` that formats the output as an ANOVA table. Behind the scenes, it fits a linear model with dummy variables.

## 18.6 Checking ANOVA Assumptions

ANOVA inherits the same assumptions as regression: independence, normality within each group, and equal variances (homoscedasticity) across groups.

### 18.6.1 Normality within Groups


``` r
penguins[, resid := residuals(fit_lm)]
```

### 18.6.2 Homogeneity of Variance -- Levene's Test from Scratch

Levene's test applies ANOVA to the **absolute deviations** from group medians:


``` r
# Levene's test from scratch
penguins[, group_median := median(flipper_length_mm), by = species]
penguins[, abs_dev := abs(flipper_length_mm - group_median)]

fit_levene <- lm(abs_dev ~ species, data = penguins)
levene_anova <- anova(fit_levene)

c(
  "Levene F" = levene_anova["species", "F value"],
  "p-value" = levene_anova["species", "Pr(>F)"]
)
```

```
 Levene F   p-value 
0.3305621 0.7187511 
```

A non-significant result suggests the equal variance assumption is reasonable.
