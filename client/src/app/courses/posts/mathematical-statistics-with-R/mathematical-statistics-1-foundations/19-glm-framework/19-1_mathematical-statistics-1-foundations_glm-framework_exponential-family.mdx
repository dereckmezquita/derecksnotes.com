# Chapter 19 -- GLM Framework

## Part 1: The Exponential Family

Up to this point every model we have fitted assumes a continuous, normally distributed response. But the real world is not so obliging. A patient either survives or does not. A website visitor clicks or does not click. A count of insurance claims cannot be negative or fractional. Forcing these outcomes into the straight-jacket of ordinary linear regression produces nonsensical predictions -- probabilities outside $[0, 1]$, negative counts -- and violates every assumption that makes inference valid.

This chapter develops the **Generalised Linear Model (GLM)**, a framework that extends linear regression to responses from any distribution in the *exponential family*. In this first part we ask *why* generalisation is necessary, define the exponential family precisely, and show that every distribution we have already studied -- Normal, Binomial, Poisson, Gamma -- is a member.

---

## 19.1 Why Generalise?

### 19.1.1 Linear Regression Fails for Binary Data

Suppose we want to model whether a baby has low birth weight ($< 2500\text{g}$) as a function of the mother's weight. The response is binary: $Y_i \in \{0, 1\}$. Let us see what happens when we fit an ordinary linear model.

## 19.2 The Exponential Family of Distributions

### 19.2.1 One-Parameter Canonical Form

A random variable $Y$ belongs to the **one-parameter exponential family** if its density (or mass function) can be written as:

$$f(y \mid \theta) = h(y) \exp\!\bigl[\eta(\theta)\, T(y) - A(\theta)\bigr]$$

where:

- $\theta$ is the **natural (canonical) parameter**.
- $T(y)$ is the **sufficient statistic**.
- $A(\theta)$ is the **log-partition function** (ensures the density integrates to 1).
- $h(y)$ is the **base measure** (does not depend on $\theta$).

When $T(y) = y$ and $\eta(\theta) = \theta$, we say the family is in **canonical form**:

$$f(y \mid \theta) = h(y) \exp\!\bigl[\theta\, y - A(\theta)\bigr]$$

This compact representation is the engine of the entire GLM framework. Every result we derive in this chapter flows from this single equation.

### 19.2.2 Why This Form Matters

The exponential family is not merely an algebraic curiosity. It has three properties that make it central to statistics:

1. **Sufficient statistics exist.** The entire sample information about $\theta$ is captured by $\sum T(y_i)$.
2. **Conjugate priors exist.** Bayesian updating is analytically tractable.
3. **Mean and variance follow from $A(\theta)$.** We will derive this next.

---

## 19.3 Mean and Variance from the Log-Partition Function

**Theorem.** If $Y$ belongs to the canonical exponential family $f(y \mid \theta) = h(y)\exp[\theta y - A(\theta)]$, then:

$$E[Y] = A'(\theta), \qquad \text{Var}(Y) = A''(\theta).$$

**Proof.** Since $\int f(y \mid \theta)\, dy = 1$, we have:

$$\int h(y) \exp[\theta y - A(\theta)]\, dy = 1$$

Differentiate both sides with respect to $\theta$:

$$\int h(y) [y - A'(\theta)] \exp[\theta y - A(\theta)]\, dy = 0$$

$$\Rightarrow \int y\, f(y \mid \theta)\, dy = A'(\theta) \cdot \int f(y \mid \theta)\, dy$$

$$\Rightarrow E[Y] = A'(\theta).$$

Differentiate again:

$$\int h(y) \bigl\{[y - A'(\theta)]^2 - A''(\theta)\bigr\} \exp[\theta y - A(\theta)]\, dy = 0$$

$$\Rightarrow E[(Y - A'(\theta))^2] = A''(\theta)$$

$$\Rightarrow \text{Var}(Y) = A''(\theta). \qquad \square$$

This is remarkable: the entire moment structure of an exponential family distribution is encoded in a single function $A(\theta)$.

## 19.4 Showing Classic Distributions Are Members

### 19.4.1 The Normal Distribution

The $N(\mu, \sigma^2)$ density (treating $\sigma^2$ as known) is:

$$f(y \mid \mu) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left[-\frac{(y - \mu)^2}{2\sigma^2}\right]$$

Expand the square:

$$= \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left[-\frac{y^2}{2\sigma^2}\right] \exp\!\left[\frac{\mu}{\sigma^2} y - \frac{\mu^2}{2\sigma^2}\right]$$

Identify the pieces:

- $\theta = \mu / \sigma^2$ (canonical parameter)
- $T(y) = y$
- $A(\theta) = \mu^2 / (2\sigma^2) = \sigma^2 \theta^2 / 2$
- $h(y) = (2\pi\sigma^2)^{-1/2} \exp[-y^2 / (2\sigma^2)]$

Check: $A'(\theta) = \sigma^2 \theta = \mu = E[Y]$. And $A''(\theta) = \sigma^2 = \text{Var}(Y)$. Correct.

### 19.4.2 The Bernoulli (Binomial with $n = 1$)

$$P(Y = y \mid p) = p^y (1-p)^{1-y} = \exp\!\left[y \log\frac{p}{1-p} + \log(1-p)\right]$$

- $\theta = \log[p / (1-p)]$ (the **log-odds** or **logit**)
- $A(\theta) = -\log(1-p) = \log(1 + e^\theta)$
- $h(y) = 1$

Check: $A'(\theta) = e^\theta / (1 + e^\theta) = p = E[Y]$. And $A''(\theta) = p(1-p) = \text{Var}(Y)$. Correct.

### 19.4.3 The Poisson Distribution

$$P(Y = y \mid \lambda) = \frac{\lambda^y e^{-\lambda}}{y!} = \frac{1}{y!} \exp[y \log\lambda - \lambda]$$

- $\theta = \log\lambda$
- $A(\theta) = \lambda = e^\theta$
- $h(y) = 1/y!$

Check: $A'(\theta) = e^\theta = \lambda = E[Y]$. And $A''(\theta) = e^\theta = \lambda = \text{Var}(Y)$. Correct. (For Poisson, mean equals variance -- a property we shall revisit in Chapter 21.)

### 19.4.4 The Gamma Distribution

The Gamma density with known shape $\alpha$ and rate parameter $\beta$ is:

$$f(y \mid \beta) = \frac{\beta^\alpha}{\Gamma(\alpha)} y^{\alpha - 1} e^{-\beta y}, \quad y > 0$$

Rewrite:

$$= \frac{y^{\alpha - 1}}{\Gamma(\alpha)} \exp[-\beta y + \alpha \log \beta]$$

- $\theta = -\beta$
- $A(\theta) = -\alpha \log(-\theta) = \alpha \log \beta$
- $h(y) = y^{\alpha - 1} / \Gamma(\alpha)$

Check: $A'(\theta) = -\alpha / \theta = \alpha / \beta = E[Y]$. And $A''(\theta) = \alpha / \theta^2 = \alpha / \beta^2 = \text{Var}(Y)$. Correct.

## 19.7 The Dispersion Parameter

The full exponential dispersion family generalises the canonical form to include a **dispersion parameter** $\phi$:

$$f(y \mid \theta, \phi) = h(y, \phi) \exp\!\left[\frac{\theta y - A(\theta)}{\phi}\right]$$

For the Normal distribution, $\phi = \sigma^2$. For Bernoulli and Poisson, $\phi = 1$ (no dispersion parameter). For Gamma with known shape $\alpha$, $\phi = 1/\alpha$.

The dispersion parameter governs the *spread* of the distribution for a given mean. When $\phi > 1$ we have **overdispersion** -- more variability than the baseline model predicts. This concept will be critical in Chapter 21 when we encounter count data with variance exceeding the mean.

## 19.8 Sufficiency and Maximum Likelihood

### 19.8.1 The Log-Likelihood

Given independent observations $y_1, \ldots, y_n$ from a canonical exponential family, the log-likelihood is:

$$\ell(\theta) = \sum_{i=1}^n \bigl[\theta\, y_i - A(\theta)\bigr] + \sum_{i=1}^n \log h(y_i)$$

The second sum does not depend on $\theta$, so the MLE depends on the data only through $\sum y_i$ -- confirming that $\sum y_i$ is a sufficient statistic.

### 19.8.2 The Score and Fisher Information

The score function is:

$$U(\theta) = \frac{d\ell}{d\theta} = \sum_{i=1}^n [y_i - A'(\theta)] = \sum_{i=1}^n [y_i - \mu]$$

Setting $U(\theta) = 0$ gives $\hat{\mu} = \bar{y}$, the sample mean -- exactly as we should expect.

The Fisher information is:

$$I(\theta) = -E\!\left[\frac{d^2\ell}{d\theta^2}\right] = n\, A''(\theta) = n\, \text{Var}(Y)$$

This connects the curvature of the log-likelihood to the variance of the distribution -- a relationship we exploit in the IRLS algorithm of Part 2.


``` r
set.seed(42)
```

## 19.9 From Exponential Family to GLM: A Preview

The three components of a GLM are:

1. **Random component**: $Y_i$ follows an exponential family distribution.
2. **Systematic component**: a linear predictor $\eta_i = \mathbf{x}_i^\top \boldsymbol{\beta}$.
3. **Link function**: $g(\mu_i) = \eta_i$, connecting the mean to the linear predictor.

The **canonical link** sets $g(\mu) = \theta$, the natural parameter. For Bernoulli this is the logit; for Poisson this is the log; for Normal this is the identity; for Gamma this is the reciprocal. In Part 2 we develop the link function and the iterative algorithm for fitting GLMs.

## 19.10 Building Intuition: Exponential Family as Tilted Distributions

One can think of the exponential family as taking a "base" distribution $h(y)$ and *tilting* it by $\exp(\theta y)$, then renormalising via $\exp(-A(\theta))$. Increasing $\theta$ puts more weight on larger values of $y$; decreasing $\theta$ concentrates probability on smaller values.


``` r
y_grid <- seq(-5, 10, length.out = 400)
base <- dnorm(y_grid, 0, 1)

tilts <- c(-1, 0, 1, 2)
tilt_dt <- data.table$rbindlist(lapply(tilts, function(th) {
  unnorm <- base * exp(th * y_grid)
  Z <- sum(unnorm) * diff(y_grid[1:2])
  data.table$data.table(y = y_grid, density = unnorm / Z, theta = paste0("theta = ", th))
}))

ggplot2$ggplot(tilt_dt, ggplot2$aes(x = y, y = density, colour = theta)) +
  ggplot2$geom_line(linewidth = 0.9) +
  ggplot2$labs(
    x = "y", y = "Density", colour = "Canonical parameter",
    title = "Tilting a standard Normal base measure"
  ) +
  ggplot2$theme_minimal(base_size = 13) +
  ggplot2$theme(legend.position = "bottom")
```

## Communicating to Stakeholders

When explaining the exponential family and the need for GLMs:

- **Start with the failure**: show the OLS-on-binary plot. Everyone can see that predictions outside $[0, 1]$ are nonsensical. This motivates the entire framework without any algebra.
- **Use the word "family"**: explain that Normal, Binomial, and Poisson distributions are all "members of the same family", much like different breeds of dog share a common ancestor.
- **Emphasise the practical payoff**: "GLMs let us use the right distribution for the data -- binary outcomes get logistic regression, counts get Poisson regression -- while keeping the familiar structure of a linear model."
- **Avoid the term "canonical parameter"** unless speaking to statisticians. Say "the natural scale" or "the transformed scale" instead.
