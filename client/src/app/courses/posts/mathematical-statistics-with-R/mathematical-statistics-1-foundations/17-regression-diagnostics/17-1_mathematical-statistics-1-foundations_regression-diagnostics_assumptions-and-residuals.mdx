# Chapter 17 -- Regression Diagnostics

## Part 1: Assumptions and Residuals

Chapters 15 and 16 derived the OLS estimator and showed that it is BLUE (Best Linear Unbiased Estimator) under the Gauss-Markov assumptions. But every theorem has premises, and the validity of confidence intervals, $p$-values, and prediction intervals depends entirely on whether those premises hold. In this part we state the assumptions precisely, develop residual-based diagnostics to check each one, and rebuild the four standard `plot(lm())` diagnostics from scratch in ggplot2.

---

## 17.1 The Four Assumptions -- Stated Precisely

Write the linear model as:

$$Y_i = \mathbf{x}_i'\boldsymbol{\beta} + \varepsilon_i, \quad i = 1, \ldots, n$$

The classical assumptions are:

### 17.1.1 (A1) Linearity

The conditional expectation of $Y$ given $\mathbf{x}$ is a linear function of the parameters:

$$E[Y_i \mid \mathbf{x}_i] = \mathbf{x}_i'\boldsymbol{\beta}$$

This does **not** require that $Y$ be a linear function of the raw predictors -- $Y = \beta_0 + \beta_1 x + \beta_2 x^2 + \varepsilon$ is perfectly linear *in the parameters*. The assumption fails when the true relationship involves interactions or non-linear functions of $\boldsymbol{\beta}$ that are not captured by the model.

### 17.1.2 (A2) Independence

The errors $\varepsilon_1, \ldots, \varepsilon_n$ are mutually independent:

$$\text{Cov}(\varepsilon_i, \varepsilon_j) = 0 \quad \text{for all } i \neq j$$

In matrix notation: $\text{Cov}(\boldsymbol{\varepsilon})$ is diagonal. This assumption is violated by time-series data (serial correlation), clustered data (observations within the same group are correlated), and spatial data.

### 17.1.3 (A3) Homoscedasticity

The variance of each error is the same constant $\sigma^2$:

$$\text{Var}(\varepsilon_i \mid \mathbf{x}_i) = \sigma^2 \quad \text{for all } i$$

Combined with (A2), this gives $\text{Cov}(\boldsymbol{\varepsilon}) = \sigma^2 \mathbf{I}_n$. Heteroscedasticity (non-constant variance) does not bias $\hat{\boldsymbol{\beta}}$, but it invalidates the usual standard errors.

### 17.1.4 (A4) Normality

The errors follow a normal distribution:

$$\varepsilon_i \sim N(0, \sigma^2)$$

This is the strongest assumption and is only needed for exact $t$ and $F$ inference in finite samples. When $n$ is large, the Central Limit Theorem ensures that $\hat{\boldsymbol{\beta}}$ is approximately normal regardless of the error distribution (see Part 2).

### 17.1.5 Summary Table

| Assumption | Mathematical statement | What breaks if violated |
|:-----------|:----------------------|:------------------------|
| Linearity | $E[Y \mid \mathbf{x}] = \mathbf{x}'\boldsymbol{\beta}$ | $\hat{\boldsymbol{\beta}}$ is biased |
| Independence | $\text{Cov}(\varepsilon_i, \varepsilon_j) = 0$ | SEs wrong, $p$-values wrong |
| Homoscedasticity | $\text{Var}(\varepsilon_i) = \sigma^2$ | SEs wrong, efficiency lost |
| Normality | $\varepsilon_i \sim N(0, \sigma^2)$ | Finite-sample inference inexact |

## 17.2 Residuals -- The Diagnostic Currency

### 17.2.1 Raw Residuals

Given the fitted model $\hat{\mathbf{Y}} = \mathbf{H}\mathbf{Y}$, the residual vector is:

$$\mathbf{e} = \mathbf{Y} - \hat{\mathbf{Y}} = (\mathbf{I} - \mathbf{H})\mathbf{Y}$$

The raw residual $e_i$ estimates the unobservable error $\varepsilon_i$, but with a crucial difference: the residuals have **unequal variances** even when the true errors do not.

### 17.2.2 Variance of Residuals

Since $\mathbf{e} = (\mathbf{I} - \mathbf{H})\boldsymbol{\varepsilon}$ (under the true model):

$$\text{Cov}(\mathbf{e}) = \sigma^2(\mathbf{I} - \mathbf{H})$$

Therefore:

$$\text{Var}(e_i) = \sigma^2(1 - h_{ii})$$

where $h_{ii}$ is the $i$th diagonal element of the hat matrix $\mathbf{H}$. Points with high leverage ($h_{ii}$ close to 1) have residuals that are artificially shrunk towards zero -- the model is forced to pass close to high-leverage points.

### 17.2.3 Standardised (Internally Studentised) Residuals

To give every residual the same scale, we standardise:

$$r_i = \frac{e_i}{\hat{\sigma}\sqrt{1 - h_{ii}}}$$

where $\hat{\sigma} = \sqrt{\text{MSE}} = \sqrt{\text{SSE}/(n-p)}$. Under the model assumptions, $r_i$ has approximate mean 0 and approximate variance 1.

### 17.2.4 Externally Studentised (Deleted) Residuals

A more refined version uses the leave-one-out estimate $\hat{\sigma}_{(i)}$ computed without observation $i$:

$$t_i = \frac{e_i}{\hat{\sigma}_{(i)}\sqrt{1 - h_{ii}}}$$

Under the assumptions, $t_i \sim t_{n-p-1}$ exactly. This gives a genuine $t$-test for whether observation $i$ is an outlier.

- **A funnel shape** (spread increases with fitted values): suggests heteroscedasticity (A3 violated).

## 17.4 Diagnostic Plot 2: Normal Q-Q Plot

This plot checks **(A4) normality** of the residuals.

### 17.4.1 How a Q-Q Plot Works

The idea is simple: if $Z_1, \ldots, Z_n$ are a sample from $N(0,1)$, and we sort them as $Z_{(1)} \le \cdots \le Z_{(n)}$, then $Z_{(i)}$ should be close to the $i$th theoretical quantile $\Phi^{-1}(p_i)$, where $p_i = (i - 0.5)/n$ (or a similar plotting position).

For regression, we plot the sorted standardised residuals $r_{(i)}$ against $\Phi^{-1}(p_i)$. If the residuals are normally distributed, the points fall on a straight line.

**Deviations:**
- **S-shape**: heavy tails (leptokurtic) -- more extreme values than expected.
- **Reverse S-shape**: light tails (platykurtic).
- **Curvature at one end**: skewness.

### 17.4.2 Building a Q-Q Plot from Scratch


``` r
n <- nrow(diag_dt)
diag_dt[, sorted_std := sort(standardised)]
diag_dt[, theoretical := qnorm(ppoints(n))]

# Reference line through 1st and 3rd quartiles
q_sample <- quantile(diag_dt$standardised, c(0.25, 0.75))
q_theory <- qnorm(c(0.25, 0.75))
slope_qq <- diff(q_sample) / diff(q_theory)
int_qq <- q_sample[1] - slope_qq * q_theory[1]

ggplot2$ggplot(diag_dt, ggplot2$aes(x = theoretical, y = sorted_std)) +
  ggplot2$geom_abline(
    intercept = int_qq, slope = slope_qq,
    colour = "#D55E00", linewidth = 0.8
  ) +
  ggplot2$geom_point(colour = "#0072B2", alpha = 0.5, size = 1.5) +
  ggplot2$labs(
    x = "Theoretical quantiles",
    y = "Standardised residuals",
    title = "Normal Q-Q plot (from scratch)"
  ) +
  ggplot2$theme_minimal(base_size = 13)
```

## 17.5 Diagnostic Plot 3: Scale-Location

This plot checks **(A3) homoscedasticity** specifically. It plots $\sqrt{|r_i|}$ against $\hat{y}_i$.

### 17.5.1 Why the Square Root of Absolute Residuals?

Under homoscedasticity:

$$E[|r_i|] \approx \sqrt{2/\pi} \approx 0.798$$

for all $i$ (using the fact that $|Z|$ has a half-normal distribution when $Z \sim N(0,1)$). If we plot $\sqrt{|r_i|}$ against $\hat{y}_i$, a horizontal smooth trend indicates constant variance. An upward trend indicates increasing variance.

### 17.5.2 ggplot2 Recreation


``` r
diag_dt[, sqrt_abs_std := sqrt(abs(standardised))]

ggplot2$ggplot(diag_dt, ggplot2$aes(x = fitted, y = sqrt_abs_std)) +
  ggplot2$geom_point(colour = "#0072B2", alpha = 0.4, size = 1.5) +
  ggplot2$geom_smooth(
    method = "loess", formula = y ~ x,
    colour = "#D55E00", se = FALSE, linewidth = 0.9
  ) +
  ggplot2$labs(
    x = "Fitted values",
    y = expression(sqrt("|Standardised residuals|")),
    title = "Scale-Location plot"
  ) +
  ggplot2$theme_minimal(base_size = 13)
```

## 17.6 Diagnostic Plot 4: Residuals vs Leverage

This plot identifies **influential observations** -- points that have an outsized effect on the fitted regression.

### 17.6.1 Leverage Recap

The leverage $h_{ii}$ measures how far observation $i$'s predictor values are from the centre of the predictor space. We showed in Chapter 16 that:

$$h_{ii} = \mathbf{x}_i'(\mathbf{X}'\mathbf{X})^{-1}\mathbf{x}_i$$

Properties:
- $0 \le h_{ii} \le 1$
- $\sum_{i=1}^n h_{ii} = p$ (the number of parameters)
- Average leverage: $\bar{h} = p/n$
- Rule of thumb: $h_{ii} > 2p/n$ is "high leverage"

### 17.6.2 Cook's Distance Preview

The fourth plot overlays contours of Cook's distance (derived fully in Part 2). For now, note that Cook's distance combines leverage and residual size:

$$D_i = \frac{r_i^2}{p} \cdot \frac{h_{ii}}{1 - h_{ii}}$$

Points with large $D_i$ are simultaneously far from the centre of the predictor space and poorly fitted.

### 17.6.3 ggplot2 Recreation


``` r
diag_dt[, cooks_d := cooks.distance(fit_boston)]

p_params <- length(coef(fit_boston))
leverage_cutoff <- 2 * p_params / n

ggplot2$ggplot(diag_dt, ggplot2$aes(x = leverage, y = standardised)) +
  ggplot2$geom_point(
    ggplot2$aes(size = cooks_d),
    colour = "#0072B2", alpha = 0.4
  ) +
  ggplot2$geom_hline(yintercept = 0, linetype = "dashed", colour = "grey50") +
  ggplot2$geom_vline(
    xintercept = leverage_cutoff,

    method = "loess", formula = y ~ x,
    colour = "#D55E00", se = FALSE, linewidth = 0.9
  ) +
  ggplot2$scale_size_continuous(range = c(0.5, 5), name = "Cook's D") +
  ggplot2$labs(
    x = "Leverage",
```

## 17.10 Formal Tests for Assumptions

Graphical diagnostics are the primary tool, but formal tests provide a complementary check.

### 17.10.1 Shapiro-Wilk Test for Normality


``` r
# Shapiro-Wilk on a subsample (test limited to n <= 5000)
set.seed(42)
idx <- sample(nrow(boston), min(nrow(boston), 5000))
shapiro.test(residuals(fit_boston)[idx])
```

```

	Shapiro-Wilk normality test

data:  residuals(fit_boston)[idx]
W = 0.91406, p-value = 2.385e-16
```

### 17.10.2 Breusch-Pagan Test for Heteroscedasticity

The Breusch-Pagan test regresses the squared residuals on the predictors. Under homoscedasticity, $nR^2 \sim \chi^2_k$ where $k$ is the number of predictors.


``` r
e2 <- residuals(fit_boston)^2
bp_fit <- lm(e2 ~ lstat + rm + age, data = boston)
bp_stat <- nrow(boston) * summary(bp_fit)$r.squared
bp_df <- length(coef(bp_fit)) - 1

c(
  "BP statistic" = bp_stat,
  "df" = bp_df,
  "p-value" = pchisq(bp_stat, df = bp_df, lower.tail = FALSE)
)
```

```
BP statistic           df      p-value 
1.977069e+01 3.000000e+00 1.893656e-04 
```

The highly significant result confirms what the scale-location plot suggested: the Boston housing model exhibits heteroscedasticity.

### 17.10.3 Durbin-Watson for Autocorrelation


``` r
e_bos <- residuals(fit_boston)
DW <- sum(diff(e_bos)^2) / sum(e_bos^2)
c("Durbin-Watson" = DW)
```

```
Durbin-Watson 
    0.8507235 
```

A value close to 2 indicates no first-order autocorrelation. Spatial correlation (observations that are close geographically have similar residuals) is a real concern in the Boston data but requires specialised tests.

## Communicating to Stakeholders

When presenting regression diagnostics:

- **Lead with the plots, not the tests.** A residual-vs-fitted plot tells a richer story than a single $p$-value. Show the four-panel diagnostic display and explain what each panel checks.
- **Translate the jargon.** "The funnel shape in the residuals tells us the model's predictions are less reliable for expensive homes than for cheap homes."
- **Be honest about violations.** If the diagnostics reveal problems, say so -- and say what you plan to do about it (transformations, robust standard errors, different model). A model with acknowledged limitations is more trustworthy than one presented without caveats.
- **Context matters.** Mild non-normality with $n = 5{,}000$ is far less concerning than the same violation with $n = 30$.

## Quick Reference

| Diagnostic | Checks | Pattern under violation |
|:-----------|:-------|:----------------------|
| Residuals vs Fitted | Linearity + homoscedasticity | Curve = non-linear; funnel = heteroscedastic |
| Normal Q-Q | Normality of errors | S-shape = heavy tails; curve = skew |
| Scale-Location | Homoscedasticity | Upward trend = increasing variance |
| Residuals vs Leverage | Influential points | Large residual + high leverage = influential |
| Shapiro-Wilk | Normality (formal) | Small $p$ = non-normal |
| Breusch-Pagan | Homoscedasticity (formal) | Small $p$ = heteroscedastic |
| Durbin-Watson | Autocorrelation | $\ll 2$ = positive; $\gg 2$ = negative |
