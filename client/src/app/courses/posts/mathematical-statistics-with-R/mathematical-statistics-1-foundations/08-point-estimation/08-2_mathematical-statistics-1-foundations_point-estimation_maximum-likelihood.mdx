# Chapter 8 -- Point Estimation

## Part 2: Maximum Likelihood Estimation

The method of moments (Part 1) is intuitive and often yields quick answers. But it ignores the actual *shape* of the data's distribution -- it only matches a few summary statistics. **Maximum Likelihood Estimation (MLE)** takes a fundamentally different approach: it asks, "which parameter value makes the observed data most probable?"

This simple question leads to an estimation framework that dominates modern statistics and machine learning alike. MLEs enjoy a suite of remarkable theoretical properties and, in large samples, are essentially the best estimators available.

---

## 8.6 The Likelihood Function

### 8.6.1 Definition

Let $X_1, \ldots, X_n$ be iid with density (or PMF) $f(x; \theta)$. The **likelihood function** is:

$$L(\theta) = L(\theta; x_1, \ldots, x_n) = \prod_{i=1}^n f(x_i; \theta).$$

This is the same expression as the joint density of the sample, but we view it as a function of $\theta$ with the data $x_1, \ldots, x_n$ held fixed.

> **Crucial perspective shift.** The density $f(x;\theta)$ asks: "Given the parameter, how likely is a particular observation?" The likelihood $L(\theta)$ flips this: "Given the data, how plausible is each parameter value?"

### 8.6.2 The Log-Likelihood

Products of many small numbers are numerically unstable and analytically awkward. Since $\log$ is a strictly increasing function, maximising $L(\theta)$ is equivalent to maximising:

$$\ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log f(x_i; \theta).$$

This converts products into sums, which are far easier to differentiate.

### 8.6.3 The Maximum Likelihood Estimator

The **MLE** $\hat\theta_{\text{ML}}$ is the value of $\theta$ that maximises $L(\theta)$ (or equivalently $\ell(\theta)$):

$$\hat\theta_{\text{ML}} = \arg\max_\theta \; \ell(\theta).$$

In most regular problems, we find it by solving the **score equation**:

$$\frac{d\ell}{d\theta} = 0,$$

and checking the second derivative is negative.

---

## 8.7 Deriving MLEs for Common Distributions

### 8.7.1 MLE for Bernoulli($p$)

Let $X_1, \ldots, X_n \sim \text{Bernoulli}(p)$. The PMF is $f(x;p) = p^x(1-p)^{1-x}$.

**Log-likelihood:**

$$\ell(p) = \sum_{i=1}^n [x_i \log p + (1 - x_i)\log(1-p)] = s\log p + (n-s)\log(1-p),$$

where $s = \sum_{i=1}^n x_i$.

**Score equation:**

$$\frac{d\ell}{dp} = \frac{s}{p} - \frac{n-s}{1-p} = 0.$$

Solving: $s(1-p) = (n-s)p$, hence $s = np$, giving:

$$\boxed{\hat p_{\text{ML}} = \frac{s}{n} = \bar X.}$$

The MLE coincides with the MOM estimator and the intuitive sample proportion.

### 8.7.2 MLE for Poisson($\lambda$)

Let $X_1, \ldots, X_n \sim \text{Poisson}(\lambda)$. The PMF is $f(x;\lambda) = e^{-\lambda}\lambda^x / x!$.

**Log-likelihood:**

$$\ell(\lambda) = \sum_{i=1}^n [-\lambda + x_i\log\lambda - \log(x_i!)] = -n\lambda + s\log\lambda - \sum\log(x_i!),$$

where $s = \sum x_i$.

**Score equation:**

$$\frac{d\ell}{d\lambda} = -n + \frac{s}{\lambda} = 0 \quad \Rightarrow \quad \boxed{\hat\lambda_{\text{ML}} = \frac{s}{n} = \bar X.}$$

### 8.7.3 MLE for Exponential($\lambda$)

Let $X_1, \ldots, X_n \sim \text{Exp}(\lambda)$ with density $f(x;\lambda) = \lambda e^{-\lambda x}$ for $x > 0$.

**Log-likelihood:**

$$\ell(\lambda) = \sum_{i=1}^n [\log\lambda - \lambda x_i] = n\log\lambda - \lambda\sum x_i.$$

**Score equation:**

$$\frac{d\ell}{d\lambda} = \frac{n}{\lambda} - \sum x_i = 0 \quad \Rightarrow \quad \boxed{\hat\lambda_{\text{ML}} = \frac{n}{\sum x_i} = \frac{1}{\bar X}.}$$

### 8.7.4 MLE for Normal($\mu$, $\sigma^2$)

This is the most important example. Let $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$.

**Log-likelihood:**

$$\ell(\mu, \sigma^2) = -\frac{n}{2}\log(2\pi) - \frac{n}{2}\log\sigma^2 - \frac{1}{2\sigma^2}\sum_{i=1}^n(x_i - \mu)^2.$$

**Partial with respect to $\mu$:**

$$\frac{\partial\ell}{\partial\mu} = \frac{1}{\sigma^2}\sum_{i=1}^n(x_i - \mu) = 0 \quad \Rightarrow \quad \boxed{\hat\mu_{\text{ML}} = \bar X.}$$

**Partial with respect to $\sigma^2$:**

$$\frac{\partial\ell}{\partial\sigma^2} = -\frac{n}{2\sigma^2} + \frac{1}{2(\sigma^2)^2}\sum_{i=1}^n(x_i - \mu)^2 = 0.$$

Substituting $\hat\mu = \bar X$ and solving:

$$\boxed{\hat\sigma^2_{\text{ML}} = \frac{1}{n}\sum_{i=1}^n(X_i - \bar X)^2.}$$

Note that the MLE for $\sigma^2$ divides by $n$, not $n-1$ -- it is the *biased* variance estimator. This illustrates that MLEs are not always unbiased. However, the bias vanishes as $n \to \infty$.

## 8.8 MLE from Scratch with `optim()`

When closed-form solutions are unavailable, we maximise the log-likelihood numerically.


``` r
set.seed(42)
x_gamma <- rgamma(200, shape = 3, rate = 0.5)

# Negative log-likelihood for Gamma
neg_loglik_gamma <- function(par, data) {
  alpha <- par[1]
  beta <- par[2]   # rate
  if (alpha <= 0 || beta <= 0) return(Inf)
  -sum(dgamma(data, shape = alpha, rate = beta, log = TRUE))
}

result <- optim(
  par = c(1, 1),
  fn = neg_loglik_gamma,
  data = x_gamma,
  method = "L-BFGS-B",
  lower = c(1e-6, 1e-6)
)

data.frame(
  parameter = c("alpha (shape)", "beta (rate)"),
  true_value = c(3, 0.5),
  mle = round(result$par, 4)
)
```

```
2   beta (rate)        0.5 0.4957
```

## 8.9 MLE Properties

### 8.9.1 Consistency

Under regularity conditions (the parameter space is an open set, the model is identifiable, the support of $f$ does not depend on $\theta$, and $f$ is sufficiently smooth), the MLE is consistent:

$$\hat\theta_{\text{ML}} \xrightarrow{P} \theta_0 \quad \text{as } n \to \infty.$$

### 8.9.2 Asymptotic Normality

Under the same regularity conditions, the MLE is asymptotically normal:

$$\sqrt{n}(\hat\theta_{\text{ML}} - \theta_0) \xrightarrow{d} N\!\left(0, \frac{1}{I(\theta_0)}\right),$$

where $I(\theta_0)$ is the **Fisher information** (to be studied in Part 3). In practical terms, for large $n$:

$$\hat\theta_{\text{ML}} \;\dot\sim\; N\!\left(\theta_0, \frac{1}{nI(\theta_0)}\right).$$

### 8.9.3 Asymptotic Efficiency

The MLE achieves the Cramér--Rao lower bound asymptotically. No consistent estimator can have a smaller asymptotic variance. This makes MLE asymptotically efficient.

### 8.9.4 Invariance Property

If $\hat\theta$ is the MLE of $\theta$, then $g(\hat\theta)$ is the MLE of $g(\theta)$ for any function $g$. For example, if $\hat\sigma^2$ is the MLE of $\sigma^2$, then $\sqrt{\hat\sigma^2}$ is the MLE of $\sigma$.

## 8.12 Comparison: MOM vs. MLE


``` r
set.seed(42)
true_alpha <- 3
true_rate <- 0.5
n <- 50
n_reps <- 5000

mom_est <- numeric(n_reps)
mle_est <- numeric(n_reps)

for (i in seq_len(n_reps)) {
  x <- rgamma(n, shape = true_alpha, rate = true_rate)
  xbar <- mean(x)
  s2 <- mean((x - xbar)^2)
  mom_est[i] <- xbar^2 / s2

  res <- tryCatch(
    optim(c(1, 1), function(par) {
      if (par[1] <= 0 || par[2] <= 0) return(Inf)
      -sum(dgamma(x, shape = par[1], rate = par[2], log = TRUE))
    }, method = "L-BFGS-B", lower = c(1e-4, 1e-4)),
    error = function(e) list(par = c(NA, NA))
  )
  mle_est[i] <- res$par[1]
}

comp_dt <- data.table$data.table(
  value = c(mom_est, mle_est),
  method = rep(c("MOM", "MLE"), each = n_reps)
)

ggplot2$ggplot(comp_dt, ggplot2$aes(x = value, fill = method)) +
  ggplot2$geom_histogram(ggplot2$aes(y = ggplot2::after_stat(density)),
    bins = 60, alpha = 0.5, position = "identity") +
  ggplot2$geom_vline(xintercept = true_alpha, linetype = "dashed",
    colour = "red", linewidth = 0.8) +
  ggplot2$labs(
    x = expression(hat(alpha)),
    y = "Density",
    fill = "Method",
    title = sprintf("MOM vs. MLE for Gamma shape (true alpha = %d, n = %d)", true_alpha, n)
  ) +
  ggplot2$coord_cartesian(xlim = c(0, 8)) +
  ggplot2$theme_minimal(base_size = 13)
```

## Communicating to Stakeholders

When explaining MLE to a non-technical audience:

- Describe the MLE as "the parameter value that best explains the data we actually observed." Avoid the word "likelihood" -- it sounds too much like "probability" and causes confusion.
- The log-likelihood surface or curve is a powerful visual. Show it and point to the peak: "This is our best estimate."
- When discussing the invariance property, simply say: "If our best estimate of the variance is 25, then our best estimate of the standard deviation is 5 -- the method is self-consistent."
- Compare MOM and MLE by saying: "Method A matches summary statistics; Method B finds the parameters that best reproduce the entire shape of the data. Method B is usually more precise."

## Quick Reference

| Concept | Formula / Description |
|:--------|:----------------------|
| **Likelihood** | $L(\theta) = \prod_{i=1}^n f(x_i; \theta)$ |
| **Log-likelihood** | $\ell(\theta) = \sum_{i=1}^n \log f(x_i; \theta)$ |
| **MLE** | $\hat\theta = \arg\max_\theta \ell(\theta)$ |
| **Score equation** | $\ell'(\theta) = 0$ |
| **MLE: Bernoulli** | $\hat p = \bar X$ |
| **MLE: Poisson** | $\hat\lambda = \bar X$ |
| **MLE: Exponential** | $\hat\lambda = 1/\bar X$ |
| **MLE: Normal** | $\hat\mu = \bar X$, $\hat\sigma^2 = \frac{1}{n}\sum(X_i - \bar X)^2$ |
| **Key properties** | Consistent, asymptotically normal, asymptotically efficient, invariant |

## Exercises

**Conceptual**

1. In your own words, explain why we maximise the log-likelihood rather than the likelihood. Give both a numerical and a mathematical reason.

2. The MLE of $\sigma^2$ for the Normal distribution is biased. Does this contradict the claim that MLEs are "optimal"? Discuss.

3. What does the invariance property of MLE mean in practice? If $\hat\lambda_{\text{ML}} = 0.25$ for an Exponential model, what is the MLE of the mean?

**Proofs and Derivations**

4. Derive the MLE for $\theta$ when $X_1, \ldots, X_n \sim \text{Uniform}(0, \theta)$. (Hint: this is not solved by setting the derivative to zero.)

5. Derive the MLE for $p$ in a Geometric distribution with PMF $P(X = k) = (1-p)^{k-1}p$ for $k = 1, 2, \ldots$

6. Show that the MLE for $\lambda$ in the Exponential distribution is consistent.

**Computational**

7. Using `optim()`, implement MLE from scratch for a **Beta distribution** fitted to the penguins' rescaled bill-length data (divide by the maximum so values lie in $(0,1)$). Compare with the MOM estimators.

8. Generate data from a mixture of two Normals: $0.4 \cdot N(2, 1) + 0.6 \cdot N(7, 1.5^2)$. Plot the log-likelihood surface as a function of the two means. Discuss why mixture MLEs are difficult to find.

9. Simulate 5,000 samples of size $n = 50$ from a Poisson(4) distribution. For each, compute the MLE. Verify empirically that (a) the MLE is approximately unbiased, (b) its variance is close to $\lambda/n = 4/50 = 0.08$, and (c) the sampling distribution is approximately Normal.

10. Fit a Normal distribution to NHANES systolic blood pressure data using both MOM and MLE. Compare the fitted densities visually and compute the KL divergence from each fit to the empirical distribution.
