# Chapter 14 -- Linear Algebra for Statistics

## Part 2: Projection and Eigenvalues

In Part 1 we established the basic vocabulary of vectors, matrices, and systems. This part develops two ideas that are absolutely central to statistical theory:

1. **Orthogonal projection** -- the geometric operation that *is* ordinary least squares.
2. **Eigenvalues and eigenvectors** -- the decomposition that powers PCA, quadratic forms, and the multivariate normal.

After this chapter, the formula $\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$ will feel less like an algebraic accident and more like an inevitable geometric truth.

---

## 14.11 Orthogonality

### 14.11.1 Orthogonal Vectors

Two vectors $\mathbf{u}$ and $\mathbf{v}$ are **orthogonal** if their dot product is zero:

$$\mathbf{u}^T \mathbf{v} = 0.$$

Geometrically, orthogonal vectors are perpendicular. In statistics, orthogonality means zero correlation (for centred vectors).

### 14.11.2 Orthogonal Complement

Given a subspace $\mathcal{V}$ of $\mathbb{R}^n$, the **orthogonal complement** $\mathcal{V}^{\perp}$ is the set of all vectors orthogonal to every vector in $\mathcal{V}$:

$$\mathcal{V}^{\perp} = \{ \mathbf{w} \in \mathbb{R}^n : \mathbf{w}^T \mathbf{v} = 0 \text{ for all } \mathbf{v} \in \mathcal{V} \}.$$

Every vector $\mathbf{y} \in \mathbb{R}^n$ can be uniquely decomposed as:

$$\mathbf{y} = \underbrace{\hat{\mathbf{y}}}_{\in \mathcal{V}} + \underbrace{\mathbf{e}}_{\in \mathcal{V}^{\perp}},$$

where $\hat{\mathbf{y}}$ is the orthogonal projection of $\mathbf{y}$ onto $\mathcal{V}$ and $\mathbf{e} = \mathbf{y} - \hat{\mathbf{y}}$ is the residual. This decomposition is the geometric heart of regression.

## 14.12 Orthogonal Projection

### 14.12.1 Projection onto a Line

The simplest projection: project vector $\mathbf{y}$ onto the line spanned by vector $\mathbf{a}$. The projection is:

$$\hat{\mathbf{y}} = \frac{\mathbf{a}^T \mathbf{y}}{\mathbf{a}^T \mathbf{a}} \mathbf{a}.$$

The scalar $c = \frac{\mathbf{a}^T \mathbf{y}}{\mathbf{a}^T \mathbf{a}}$ is the coefficient, and $\hat{\mathbf{y}} = c\mathbf{a}$ is the closest point on the line to $\mathbf{y}$.


``` r
a <- c(2, 1)
y <- c(3, 4)

# Projection coefficient
c_proj <- as.numeric(t(a) %*% y) / as.numeric(t(a) %*% a)
c_proj

# Projected vector
y_hat <- c_proj * a
y_hat

# Residual
e <- y - y_hat
e

# Verify orthogonality: residual is perpendicular to a
t(a) %*% e  # Should be 0 (or very close)
```

### 14.12.2 The Projection Matrix: Deriving $\mathbf{P} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T$

Now generalise to projecting onto the **column space** of a matrix $\mathbf{X}$ ($n \times p$, full column rank). We want the vector $\hat{\mathbf{y}} \in \text{Col}(\mathbf{X})$ that is closest to $\mathbf{y}$. Since $\hat{\mathbf{y}}$ is in the column space of $\mathbf{X}$, it can be written as $\hat{\mathbf{y}} = \mathbf{X}\hat{\boldsymbol{\beta}}$ for some $\hat{\boldsymbol{\beta}}$.

The key requirement: the residual $\mathbf{e} = \mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}}$ must be orthogonal to every column of $\mathbf{X}$:

$$\mathbf{X}^T \mathbf{e} = \mathbf{0}.$$

Expanding:

$$\mathbf{X}^T (\mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}}) = \mathbf{0}$$

$$\mathbf{X}^T \mathbf{y} = \mathbf{X}^T \mathbf{X} \hat{\boldsymbol{\beta}}$$

$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}.$$

This is the OLS estimator. It is not an algebraic trick -- it is the *only* coefficient vector that makes the residual perpendicular to the column space of $\mathbf{X}$.

The **projection matrix** (also called the **hat matrix**) is:

$$\mathbf{P} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T.$$

It maps any vector $\mathbf{y}$ to its orthogonal projection onto $\text{Col}(\mathbf{X})$:

$$\hat{\mathbf{y}} = \mathbf{P}\mathbf{y}.$$

### 14.12.3 Properties of the Projection Matrix

The projection matrix $\mathbf{P}$ has remarkable properties:

1. **Symmetric:** $\mathbf{P}^T = \mathbf{P}$.
2. **Idempotent:** $\mathbf{P}^2 = \mathbf{P}$. (Projecting twice gives the same result as projecting once -- you are already in the subspace.)
3. **Trace equals rank:** $\text{tr}(\mathbf{P}) = p$ (the number of columns of $\mathbf{X}$). Since the diagonal entries $P_{ii}$ are the leverages $h_{ii}$, this means the leverages sum to $p$.
4. **The residual-maker matrix** $\mathbf{M} = \mathbf{I} - \mathbf{P}$ is also symmetric and idempotent, and $\text{tr}(\mathbf{M}) = n - p$.


``` r
data_dir <- "../../../data"
penguins <- data.table$fread(file.path(data_dir, "primary/penguins.csv"))
```

### 14.12.4 OLS Is Projection: The Geometric Picture

In simple linear regression with an intercept, the column space of $\mathbf{X}$ is a 2-dimensional plane in $\mathbb{R}^n$. The vector $\mathbf{y}$ (observed responses) lives somewhere in $\mathbb{R}^n$. The fitted values $\hat{\mathbf{y}} = \mathbf{Py}$ are the point on this plane closest to $\mathbf{y}$. The residual vector $\mathbf{e} = \mathbf{y} - \hat{\mathbf{y}}$ is perpendicular to the plane.

This explains:

- **Why $\sum e_i = 0$:** the residual $\mathbf{e}$ is orthogonal to the column of ones in $\mathbf{X}$, which means $\mathbf{1}^T\mathbf{e} = 0$.
- **Why $\sum e_i x_i = 0$:** the residual is orthogonal to *every* column of $\mathbf{X}$.
- **The Pythagorean theorem:** $\|\mathbf{y}\|^2 = \|\hat{\mathbf{y}}\|^2 + \|\mathbf{e}\|^2$, i.e., TSS = RegSS + RSS.


``` r
e <- y - y_hat

# Pythagorean theorem: ||y||^2 = ||y_hat||^2 + ||e||^2
y_centred <- y - mean(y)
y_hat_centred <- y_hat - mean(y)

tss <- sum(y_centred^2)
regss <- sum(y_hat_centred^2)
rss <- sum(e^2)

tss
regss + rss  # Should equal TSS

# R-squared
1 - rss / tss
summary(fit)$r.squared
```

## 14.13 Eigenvalues and Eigenvectors

### 14.13.1 Definition

A scalar $\lambda$ is an **eigenvalue** of a square matrix $\mathbf{A}$ if there exists a nonzero vector $\mathbf{v}$ (the **eigenvector**) such that:

$$\mathbf{A}\mathbf{v} = \lambda \mathbf{v}.$$

In words: applying $\mathbf{A}$ to $\mathbf{v}$ merely scales $\mathbf{v}$ by $\lambda$. The direction is preserved; only the magnitude changes (and possibly the sign, if $\lambda < 0$).

### 14.13.2 How to Find Eigenvalues

Rearranging: $(\mathbf{A} - \lambda \mathbf{I})\mathbf{v} = \mathbf{0}$. For a nonzero $\mathbf{v}$ to exist, the matrix $\mathbf{A} - \lambda \mathbf{I}$ must be singular:

$$\det(\mathbf{A} - \lambda \mathbf{I}) = 0.$$

This is the **characteristic equation**. For an $n \times n$ matrix, it is a polynomial of degree $n$ in $\lambda$.

### 14.13.3 Example: 2x2

Let $\mathbf{A} = \begin{pmatrix} 4 & 1 \\ 2 & 3 \end{pmatrix}$.

The characteristic equation is:

$$(4 - \lambda)(3 - \lambda) - (1)(2) = 0$$
$$\lambda^2 - 7\lambda + 10 = 0$$
$$(\lambda - 5)(\lambda - 2) = 0$$

So $\lambda_1 = 5$ and $\lambda_2 = 2$.

### 14.13.4 Implementation from Scratch (2x2 Case)


``` r
#' Eigenvalues of a 2x2 matrix -- from scratch via quadratic formula
#' @param A 2x2 matrix
#' @return list with eigenvalues and eigenvectors
my_eigen_2x2 <- function(A) {
  a <- A[1, 1]; b <- A[1, 2]
  c <- A[2, 1]; d <- A[2, 2]

  # Characteristic equation: lambda^2 - (a+d)*lambda + (ad - bc) = 0
  tr_A <- a + d
  det_A <- a * d - b * c
  disc <- tr_A^2 - 4 * det_A

  if (disc < 0) stop("Complex eigenvalues -- not handled")

  lambda1 <- (tr_A + sqrt(disc)) / 2
  lambda2 <- (tr_A - sqrt(disc)) / 2

  # Eigenvectors: solve (A - lambda*I)v = 0
  find_evec <- function(lam) {
    # From first row: (a - lam)*v1 + b*v2 = 0
    # Set v2 = 1, then v1 = -b / (a - lam)
    if (abs(a - lam) > 1e-10) {
      v <- c(-b / (a - lam), 1)
    } else {
      v <- c(1, -(c) / (d - lam))
    }
    v / sqrt(sum(v^2))  # Normalise
  }

  list(
    values = c(lambda1, lambda2),
    vectors = cbind(find_evec(lambda1), find_evec(lambda2))
  )
}

my_eigen_2x2(A)
eigen(A)
```

## 14.14 The Spectral Theorem

### 14.14.1 Statement

If $\mathbf{A}$ is a **real symmetric** matrix ($\mathbf{A} = \mathbf{A}^T$), then:

1. All eigenvalues are real.
2. Eigenvectors corresponding to distinct eigenvalues are orthogonal.
3. $\mathbf{A}$ can be decomposed as:

$$\mathbf{A} = \mathbf{Q} \boldsymbol{\Lambda} \mathbf{Q}^T,$$

where $\mathbf{Q}$ is an orthogonal matrix ($\mathbf{Q}^T\mathbf{Q} = \mathbf{I}$) whose columns are the eigenvectors, and $\boldsymbol{\Lambda} = \text{diag}(\lambda_1, \ldots, \lambda_n)$.

This is the **spectral decomposition** (or eigen-decomposition). It is one of the most important results in all of applied mathematics.

### 14.14.2 Why It Matters for Statistics

Every covariance matrix $\boldsymbol{\Sigma}$ is real and symmetric. The spectral decomposition $\boldsymbol{\Sigma} = \mathbf{Q}\boldsymbol{\Lambda}\mathbf{Q}^T$ tells us:

- The eigenvectors $\mathbf{q}_i$ are the **principal directions** of variation.
- The eigenvalues $\lambda_i$ are the **variances** along those directions.
- PCA simply rotates the data into the eigenvector coordinate system and sorts the directions by decreasing variance.


``` r
# Covariance matrix of penguin measurements

S <- cov(as.matrix(pen))
```

The proportion of total variance explained by the $k$-th principal component is:

$$\frac{\lambda_k}{\sum_{j=1}^{p} \lambda_j} = \frac{\lambda_k}{\text{tr}(\boldsymbol{\Sigma})}.$$

## 14.15 Quadratic Forms

### 14.15.1 Definition

A **quadratic form** is a scalar-valued expression of the form:

$$Q = \mathbf{x}^T \mathbf{A} \mathbf{x} = \sum_{i=1}^{n} \sum_{j=1}^{n} A_{ij} x_i x_j,$$

where $\mathbf{A}$ is a symmetric matrix.

Quadratic forms appear everywhere in statistics:

- **Sum of squares:** $\text{RSS} = \mathbf{e}^T\mathbf{e} = \mathbf{y}^T\mathbf{M}\mathbf{y}$, where $\mathbf{M} = \mathbf{I} - \mathbf{P}$.
- **Mahalanobis distance:** $D^2 = (\mathbf{x} - \boldsymbol{\mu})^T \boldsymbol{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu})$.
- **$\chi^2$ statistic:** $\chi^2 = \mathbf{z}^T\mathbf{z}$ where $\mathbf{z} \sim N(\mathbf{0}, \mathbf{I})$.

### 14.15.2 Expected Value of a Quadratic Form

If $\mathbf{X} \sim (\boldsymbol{\mu}, \boldsymbol{\Sigma})$ (any distribution with mean $\boldsymbol{\mu}$ and covariance $\boldsymbol{\Sigma}$), then:

$$E[\mathbf{X}^T \mathbf{A} \mathbf{X}] = \text{tr}(\mathbf{A}\boldsymbol{\Sigma}) + \boldsymbol{\mu}^T \mathbf{A} \boldsymbol{\mu}.$$

This result (proved using the trace trick) is fundamental for deriving the expected values of sums of squares in ANOVA.



## 14.16 Positive Definiteness

### 14.16.1 Definition

A symmetric matrix $\mathbf{A}$ is:

- **Positive definite** if $\mathbf{x}^T \mathbf{A} \mathbf{x} > 0$ for all nonzero $\mathbf{x}$.
- **Positive semi-definite** if $\mathbf{x}^T \mathbf{A} \mathbf{x} \ge 0$ for all $\mathbf{x}$.
- **Negative definite** if $\mathbf{x}^T \mathbf{A} \mathbf{x} < 0$ for all nonzero $\mathbf{x}$.

### 14.16.2 Equivalent Conditions

For a symmetric matrix $\mathbf{A}$, the following are equivalent:

1. $\mathbf{A}$ is positive definite.
2. All eigenvalues of $\mathbf{A}$ are strictly positive.
3. All leading principal minors are positive (Sylvester's criterion).
4. There exists an invertible matrix $\mathbf{L}$ such that $\mathbf{A} = \mathbf{L}\mathbf{L}^T$ (Cholesky decomposition).

### 14.16.3 Statistical Significance

Every covariance matrix $\boldsymbol{\Sigma}$ is positive semi-definite (because $\text{Var}(\mathbf{a}^T\mathbf{X}) = \mathbf{a}^T\boldsymbol{\Sigma}\mathbf{a} \ge 0$). It is strictly positive definite if and only if no linear combination of the variables is constant -- i.e., no perfect multicollinearity.

If $\boldsymbol{\Sigma}$ is positive definite, its inverse exists, and we can compute Mahalanobis distances and fit regression models. If it is only semi-definite (singular), we need regularisation (e.g., ridge regression).

## 14.17 Communicating to Stakeholders

Eigenvalues and projections are rarely discussed directly with non-technical audiences. Instead, communicate the *consequences*:

1. **PCA:** "We found the directions in which the data vary most. The first principal component captures X% of the total variation, so a single summary score retains most of the information."

2. **Multicollinearity:** "Two of the predictors are nearly measuring the same thing. This makes the model unstable -- small changes in the data cause large changes in the coefficients. We need to remove one or combine them."

3. **Mahalanobis distance:** "This measurement accounts for the fact that variables have different scales and are correlated. A point that looks close in one variable might be far away when we consider all variables together."

4. **Projection / regression:** "The model finds the prediction that is, in a precise mathematical sense, the closest possible to the observed data. The residuals are the leftover noise that the model cannot explain."

## 14.18 Quick Reference

| Concept | Formula | R code |
|:---|:---|:---|
| Eigendecomposition | $\mathbf{A} = \mathbf{Q}\boldsymbol{\Lambda}\mathbf{Q}^T$ | `eigen(A)` |
| Projection matrix | $\mathbf{P} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T$ | `X %*% solve(crossprod(X)) %*% t(X)` |
| Residual maker | $\mathbf{M} = \mathbf{I} - \mathbf{P}$ | `diag(n) - P` |
| Leverages | $h_{ii} = P_{ii}$ | `hatvalues(lm_fit)` |
| Cholesky | $\mathbf{A} = \mathbf{L}\mathbf{L}^T$ | `chol(A)` |
| Mahalanobis distance | $D^2 = (\mathbf{x}-\boldsymbol{\mu})^T\boldsymbol{\Sigma}^{-1}(\mathbf{x}-\boldsymbol{\mu})$ | `mahalanobis(x, mu, Sigma)` |
| Condition number | $\kappa = \lambda_{\max}/\lambda_{\min}$ | `kappa(A)` |

**Key identities for derivations:**

- $E[\mathbf{X}^T\mathbf{A}\mathbf{X}] = \text{tr}(\mathbf{A}\boldsymbol{\Sigma}) + \boldsymbol{\mu}^T\mathbf{A}\boldsymbol{\mu}$
- $\text{tr}(\mathbf{P}) = \text{rank}(\mathbf{X}) = p$
- $\text{tr}(\mathbf{M}) = n - p$
- If $\mathbf{P}$ is idempotent, its eigenvalues are 0 and 1 only.
