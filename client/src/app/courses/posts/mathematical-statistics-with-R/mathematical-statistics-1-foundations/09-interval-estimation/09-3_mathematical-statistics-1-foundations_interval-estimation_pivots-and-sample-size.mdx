# Chapter 9 -- Interval Estimation

## Part 3: Pivotal Quantities and Sample Size

In Parts 1 and 2 we derived confidence intervals for means, proportions, and variances using slightly different arguments each time. Now we step back and ask: *is there a single recipe that produces all of these intervals?* The answer is yes -- the method of **pivotal quantities** provides a unified framework. We then apply this framework to the practical problem of **sample-size determination**: before collecting data, how large must $n$ be to achieve a desired precision?

We close the chapter with a real-world application: estimating a treatment effect from the Streptomycin Trial for pulmonary tuberculosis.

---

## 9.9 Pivotal Quantities: The General Method

### 9.9.1 Definition

A **pivotal quantity** (or simply a *pivot*) is a function $Q(\mathbf{X}, \theta)$ of the data $\mathbf{X} = (X_1, \ldots, X_n)$ and the parameter $\theta$ whose distribution **does not depend on** $\theta$ or any other unknown parameter.

Formally, $Q$ is a pivot for $\theta$ if $Q(\mathbf{X}, \theta) \sim F$ where $F$ is a completely known distribution.

### 9.9.2 Why Pivots Work

If we can find a pivot $Q$ and known quantiles $q_L, q_U$ such that $P(q_L \le Q \le q_U) = 1 - \alpha$, then we can algebraically "invert" the inequality to isolate $\theta$, yielding a confidence interval.

The recipe:

1. **Identify a pivot** $Q(\mathbf{X}, \theta)$ with a known distribution.
2. **Choose quantiles** $q_L$ and $q_U$ such that $P(q_L \le Q \le q_U) = 1 - \alpha$.
3. **Invert** the inequality to obtain $L(\mathbf{X}) \le \theta \le U(\mathbf{X})$.

### 9.9.3 Unifying the Intervals We Already Know

| Parameter | Pivot | Known Distribution | Inversion |
|:----------|:------|:-------------------|:----------|
| $\mu$ (known $\sigma$) | $Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}}$ | $N(0,1)$ | Z-interval |
| $\mu$ (unknown $\sigma$) | $T = \frac{\bar{X} - \mu}{S/\sqrt{n}}$ | $t_{n-1}$ | t-interval |
| $\sigma^2$ (Normal) | $Q = \frac{(n-1)S^2}{\sigma^2}$ | $\chi^2_{n-1}$ | Chi-square interval |
| $p$ (proportion) | $Z = \frac{\hat{p} - p}{\sqrt{p(1-p)/n}}$ | $N(0,1)$ approx. | Wilson interval |

Every CI in this chapter was obtained by the same three steps. The only difference is the pivot and its distribution. Once you recognise this pattern, constructing a CI for a *new* parameter becomes straightforward -- provided you can find a suitable pivot.

### 9.9.4 A New Example: Exponential Rate Parameter

Let $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \text{Exp}(\lambda)$. Then $\sum X_i \sim \text{Gamma}(n, 1/\lambda)$, and equivalently $2\lambda \sum X_i \sim \chi^2_{2n}$ (since $2\lambda X_i \sim \text{Exp}(1/2) = \chi^2_2$).

So a pivot is:

$$Q = 2\lambda \sum_{i=1}^n X_i = 2\lambda n \bar{X} \sim \chi^2_{2n}.$$

Invert $P(\chi^2_L \le 2\lambda n \bar{X} \le \chi^2_U) = 1 - \alpha$:

$$\frac{\chi^2_L}{2n\bar{X}} \le \lambda \le \frac{\chi^2_U}{2n\bar{X}}.$$


``` r
set.seed(42)

#' CI for the rate parameter of an Exponential distribution
#' @param x numeric vector of Exp(lambda) observations
#' @param conf confidence level
#' @return named vector: lower, upper, lambda_hat
my_exp_rate_ci <- function(x, conf = 0.95) {
  n <- length(x)
  xbar <- mean(x)
  alpha <- 1 - conf
  chi2_lo <- qchisq(alpha / 2, df = 2 * n)
  chi2_hi <- qchisq(1 - alpha / 2, df = 2 * n)
  c(
    lower = chi2_lo / (2 * n * xbar),
    upper = chi2_hi / (2 * n * xbar),
    lambda_hat = 1 / xbar
  )
}

# Simulate Exp(lambda = 0.5) data
x_exp <- rexp(40, rate = 0.5)
my_exp_rate_ci(x_exp)
```

```
     lower      upper lambda_hat 
 0.3633290  0.6778494  0.5085687 
```

The true rate is $\lambda = 0.5$. The CI should contain it.

### 9.9.5 When No Exact Pivot Exists

Not every problem admits a clean pivot. In such cases we have several options:

- **Approximate pivots** via the CLT (as with the proportion).
- **Likelihood-based intervals** using the asymptotic Normality of the MLE (covered in Chapter 8, Part 2).
- **Bootstrap intervals** (Chapter 13).

The pivotal method remains the gold standard when available because it produces *exact* CIs (no large-sample approximation needed).

## 9.10 Sample-Size Determination

### 9.10.1 The Question

Before collecting data, we often need to decide how many observations to gather. The answer depends on three quantities:

1. **Desired margin of error** $E$: how precise do we want the estimate to be?
2. **Confidence level** $1 - \alpha$: how confident must we be?
3. **Population variability** $\sigma$ (or a preliminary estimate thereof).

### 9.10.2 Derivation for Means

From the Z-interval, the margin of error is $E = z_{\alpha/2} \cdot \sigma / \sqrt{n}$. Solve for $n$:

$$n = \left(\frac{z_{\alpha/2} \cdot \sigma}{E}\right)^2.$$

Since $n$ must be an integer, we round up: $n = \lceil (z_{\alpha/2} \cdot \sigma / E)^2 \rceil$.

### 9.10.3 Derivation for Proportions

For the Wald interval, $E = z_{\alpha/2} \sqrt{p(1-p)/n}$. Solve for $n$:

$$n = \left(\frac{z_{\alpha/2}}{E}\right)^2 p(1-p).$$

Since $p$ is unknown before the study, we use the **worst-case** value $p = 0.5$, which maximises $p(1-p) = 0.25$:

$$n = \frac{z_{\alpha/2}^2}{4E^2}.$$

### 9.10.4 Implementation


``` r
#' Required sample size for estimating a mean
#' @param E desired margin of error
#' @param sigma known or estimated population SD
#' @param conf confidence level
#' @return integer sample size (rounded up)
sample_size_mean <- function(E, sigma, conf = 0.95) {
  z <- qnorm(1 - (1 - conf) / 2)
  ceiling((z * sigma / E)^2)
}

#' Required sample size for estimating a proportion
#' @param E desired margin of error
#' @param p anticipated proportion (default 0.5 for worst-case)
#' @param conf confidence level
#' @return integer sample size (rounded up)
sample_size_prop <- function(E, p = 0.5, conf = 0.95) {
  z <- qnorm(1 - (1 - conf) / 2)
  ceiling(z^2 * p * (1 - p) / E^2)
}

# Example: mean, sigma = 10, margin of error = 1, 95% confidence
sample_size_mean(E = 1, sigma = 10, conf = 0.95)

# Example: proportion, worst-case, margin of error = 0.03
sample_size_prop(E = 0.03)
```

```
[1] 385
[1] 1068
```

For the mean example, $n = 385$ -- which matches the standard textbook answer. For a proportion with a 3-percentage-point margin, $n = 1068$, explaining why national polls typically survey about 1000 people.

### 9.10.6 Practical Considerations

- The formula for means requires a preliminary estimate of $\sigma$. Options include pilot studies, prior research, or the range rule of thumb ($\sigma \approx \text{range}/4$).
- For proportions, using $p = 0.5$ is conservative. If you have prior information that $p \approx 0.1$, the required sample size is much smaller.
- These formulae assume simple random sampling. More complex designs (stratified, clustered) require adjustments.

## 9.11 Case Study: The Streptomycin Trial for Tuberculosis

### 9.11.1 Background

The 1948 British Medical Research Council trial of streptomycin for pulmonary tuberculosis was one of the first properly randomised controlled trials. Patients were randomly assigned to streptomycin treatment or bed rest (control). One outcome measured was improvement after 6 months, classified as "considerable improvement," "moderate/slight improvement," or "deterioration/death."

We will construct a confidence interval for the difference in improvement rates between the treatment and control groups.

### 9.11.2 Data


``` r
# Streptomycin trial data (simplified)
# Source: Hill (1948), BMJ
strep_dt <- data.table$data.table(
  group = c("Streptomycin", "Control"),
  n = c(55, 52),
  improved = c(28, 14),
  deteriorated = c(14, 22)
)

strep_dt[, prop_improved := improved / n]
strep_dt
```


### 9.11.3 CI for the Difference in Proportions

For two independent proportions $\hat{p}_1$ and $\hat{p}_2$, the standard error of the difference is:

$$\text{SE}(\hat{p}_1 - \hat{p}_2) = \sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}.$$

The CI for $p_1 - p_2$ is:

$$(\hat{p}_1 - \hat{p}_2) \pm z_{\alpha/2} \cdot \text{SE}(\hat{p}_1 - \hat{p}_2).$$


``` r
#' CI for difference in two proportions
#' @param x1 successes in group 1
#' @param n1 total in group 1
#' @param x2 successes in group 2
#' @param n2 total in group 2
#' @param conf confidence level
#' @return named vector
my_prop_diff_ci <- function(x1, n1, x2, n2, conf = 0.95) {
  p1 <- x1 / n1
  p2 <- x2 / n2
  diff <- p1 - p2
  se <- sqrt(p1 * (1 - p1) / n1 + p2 * (1 - p2) / n2)
  z <- qnorm(1 - (1 - conf) / 2)
  me <- z * se
  c(lower = diff - me, upper = diff + me, diff = diff, se = se)
}

strep_ci <- my_prop_diff_ci(28, 55, 14, 52)
strep_ci
```

```
     lower      upper       diff         se 
0.06100314 0.41871714 0.23986014 0.09125525 
```

### 9.11.4 Interpretation


``` r
cat(sprintf(
  "Treatment effect: %.1f percentage points (95%% CI: %.1f to %.1f)\n",
  100 * strep_ci["diff"],
  100 * strep_ci["lower"],
  100 * strep_ci["upper"]
))
```

```
Treatment effect: 24.0 percentage points (95% CI: 6.1 to 41.9)
```

The streptomycin group had a considerably higher improvement rate. The 95% CI for the difference does not include zero, suggesting a genuine treatment benefit. The interval tells us that the treatment effect could plausibly be as small as about 5 percentage points or as large as about 37 percentage points -- a wide range reflecting the modest sample sizes of this pioneering trial.

``` r
# Add Wilson CIs for each group
wilson_strep <- my_wilson_interval <- function(x, n, conf = 0.95) {
```

## 9.12 How Large a Sample for the Next Trial?

Suppose we wish to design a follow-up trial that detects a difference as small as 15 percentage points with a margin of error of $\pm 5$ percentage points at 95% confidence. Using the worst-case $p = 0.5$:


``` r
# Sample size per group for a proportion difference
# Approximate: treat as single proportion with E = 0.05
sample_size_prop <- function(E, p = 0.5, conf = 0.95) {
  z <- qnorm(1 - (1 - conf) / 2)
  ceiling(z^2 * p * (1 - p) / E^2)
}

n_per_group <- sample_size_prop(E = 0.05, p = 0.5)
cat(sprintf("Required sample size per group: %d\n", n_per_group))
cat(sprintf("Total trial size: %d\n", 2 * n_per_group))
```

```
Required sample size per group: 385
Total trial size: 770
```

About 385 per group, or 770 total -- substantially larger than the original 107-patient trial. This is the cost of demanding greater precision.

## 9.13 When Pivots Fail: A Preview

Not all parameters admit clean pivots. Consider the **median** of a continuous distribution, or the **ratio of two variances** from non-Normal populations. In these cases, the approaches include:

- **Likelihood ratio intervals** (using Wilks' theorem; Chapter 10).
- **Bootstrap confidence intervals** (Chapter 13).
- **Bayesian credible intervals** (Chapter 22).

Each approach has its merits. The pivot method is the simplest and most transparent when it applies, which is why we emphasise it here. But recognising its limits is important -- and we will address those limits in later chapters.

## Communicating to Stakeholders

When discussing sample-size calculations and pivots:

- Frame sample size in terms of cost: "To achieve a margin of error of 2 percentage points, we need about 2400 participants. At a cost of $£50$ per participant, that is a $£120{,}000$ study."
- Use the "quadrupling rule" as a rule of thumb: "Cutting the margin of error in half means quadrupling the budget."
- For the Streptomycin trial, translate the CI into practical terms: "Streptomycin improved the proportion of patients showing considerable improvement by roughly 24 percentage points, though the true effect could be anywhere between about 5 and 37 percentage points."
- Never present a sample-size calculation without stating the assumptions (anticipated effect size, variability, confidence level). These are judgement calls, and stakeholders should understand what drives the number.

## Quick Reference

| Concept | Key Formula / Idea |
|:--------|:-------------------|
| **Pivotal quantity** | $Q(\mathbf{X}, \theta)$ with distribution free of unknowns |
| **Pivot recipe** | Find $Q$, choose quantiles, invert |
| **Sample size (mean)** | $n = \lceil (z_{\alpha/2}\,\sigma / E)^2 \rceil$ |
| **Sample size (proportion)** | $n = \lceil z_{\alpha/2}^2 \, p(1-p) / E^2 \rceil$; worst-case $p=0.5$ |
| **Quadrupling rule** | Halving $E$ quadruples $n$ |
| **Two-proportion CI** | $(\hat{p}_1 - \hat{p}_2) \pm z_{\alpha/2}\sqrt{\hat{p}_1(1-\hat{p}_1)/n_1 + \hat{p}_2(1-\hat{p}_2)/n_2}$ |
