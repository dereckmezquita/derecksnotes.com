# Chapter 11 -- Classical Tests

## Part 2: Proportions and Chi-Square

In Part 1 we derived tests for comparing population means. Now we turn to a different class of question: is a **proportion** what we claim it is? Does a set of **counts** follow a hypothesised distribution? Are two **categorical variables** independent?

These tests all grow from the same root: the CLT applied to counts and proportions. The chi-square tests, in particular, are a beautiful application of the multivariate CLT -- the sum of squared standardised quantities converges to a chi-square distribution. Let us derive each test from first principles.

---

## 11.4 The Z-Test for a Single Proportion

### 11.4.1 The Problem

We observe $X$ successes in $n$ independent Bernoulli trials. Let $\hat{p} = X/n$ be the sample proportion. We wish to test:

$$H_0: p = p_0 \quad \text{vs} \quad H_1: p \neq p_0.$$

### 11.4.2 Derivation

**Step 1.** From Chapter 4, $X \sim \text{Bin}(n, p)$. By the CLT (Chapter 7), for large $n$:

$$\hat{p} = \frac{X}{n} \stackrel{\text{approx}}{\sim} N\!\left(p, \frac{p(1-p)}{n}\right).$$

**Step 2.** Under $H_0$, the mean is $p_0$ and the variance is $p_0(1-p_0)/n$. Standardise:

$$Z = \frac{\hat{p} - p_0}{\sqrt{p_0(1 - p_0)/n}} \stackrel{\text{approx}}{\sim} N(0, 1) \quad \text{under } H_0.$$

**Step 3.** We could also square this to get a chi-square statistic with 1 degree of freedom:

$$Z^2 = \frac{(\hat{p} - p_0)^2}{p_0(1-p_0)/n} \sim \chi^2_1.$$

This connection to the chi-square distribution foreshadows the goodness-of-fit test we derive shortly.

**Rule of thumb:** The normal approximation is adequate when $np_0 \ge 5$ and $n(1-p_0) \ge 5$.

### 11.4.3 Implementation from Scratch


``` r
set.seed(42)

# Scenario: a coin is flipped 200 times and lands heads 115 times.
# Test H0: p = 0.5
n <- 200
x <- 115
p_hat <- x / n
p_0 <- 0.5

z_stat <- (p_hat - p_0) / sqrt(p_0 * (1 - p_0) / n)
p_value_z <- 2 * pnorm(abs(z_stat), lower.tail = FALSE)

data.frame(
  p_hat = p_hat,
  z_stat = round(z_stat, 4),
  p_value = round(p_value_z, 4)
)
```

```
  p_hat z_stat p_value
1 0.575 2.1213  0.0339
```

### 11.4.4 Using `prop.test()`

``` r
prop.test(x = 115, n = 200, p = 0.5)
```

Notice that `prop.test()` applies a **continuity correction** by default (adjusting $|\hat{p} - p_0|$ downward by $1/(2n)$). This correction accounts for the fact that we are approximating a discrete distribution with a continuous one. For large $n$ the correction is negligible; for small $n$ you should consider the exact binomial test instead.

## 11.5 The Exact Binomial Test

### 11.5.1 Why "Exact"?

The z-test for proportions relies on the normal approximation. When $n$ is small or $p_0$ is near 0 or 1, this approximation is poor. The **exact binomial test** uses the binomial distribution directly -- no approximation is needed.

### 11.5.2 Derivation

Under $H_0: p = p_0$, the number of successes $X \sim \text{Bin}(n, p_0)$. The two-sided p-value is:

$$\text{p-value} = \sum_{\{k : P(X=k) \le P(X=x)\}} P(X = k)$$

where $x$ is the observed count. In other words, we sum the probabilities of all outcomes that are as unlikely or more unlikely than the one observed. This is the definition of a p-value applied without any distributional approximation.

### 11.5.3 Small-Sample Example


``` r
# A treatment is given to 12 patients; 10 recover.
# Historical recovery rate is 60%. Is the treatment effective?
n_patients <- 12
recovered <- 10
p_0_hist <- 0.60

# Exact two-sided p-value (method: sum probabilities <= P(X=10))
probs <- dbinom(0:n_patients, size = n_patients, prob = p_0_hist)
p_observed <- dbinom(recovered, size = n_patients, prob = p_0_hist)
p_value_exact <- sum(probs[probs <= p_observed])

data.frame(
  observed = recovered,
  p_observed = round(p_observed, 6),
  p_value_exact = round(p_value_exact, 4)
)
```

```

1       10   0.063852        0.1408
```

## 11.6 Chi-Square Goodness-of-Fit Test

### 11.6.1 The Problem

We observe counts $O_1, O_2, \ldots, O_k$ falling into $k$ categories and wish to test whether they follow a specified distribution with probabilities $p_1, p_2, \ldots, p_k$ (where $\sum p_i = 1$).

$$H_0: P(\text{category } i) = p_i \text{ for all } i \quad \text{vs} \quad H_1: \text{at least one } P(\text{category } i) \neq p_i.$$

### 11.6.2 Derivation from the Multinomial

**Step 1.** Under $H_0$, the expected count for category $i$ is $E_i = n p_i$. By the CLT applied to each Bernoulli indicator, for large $n$:

$$\frac{O_i - E_i}{\sqrt{E_i}} \stackrel{\text{approx}}{\sim} N(0, 1).$$

**Step 2.** We might try summing the squares of these standardised values. However, the $O_i$ are not independent -- they are constrained to sum to $n$. This constraint removes one degree of freedom.

**Step 3.** It can be shown (via the multivariate CLT and a projection argument) that:

$$\boxed{X^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i} \stackrel{\text{approx}}{\sim} \chi^2_{k-1} \quad \text{under } H_0.}$$

The degrees of freedom are $k - 1$ because of the single linear constraint $\sum O_i = n$. If we also estimate $m$ parameters from the data, the degrees of freedom reduce to $k - 1 - m$.

**Intuition:** Each term $(O_i - E_i)^2 / E_i$ measures how far the observed count deviates from expectation, relative to the expected magnitude. Large values of $X^2$ indicate poor fit.

### 11.6.3 Die-Rolling Example


``` r
set.seed(42)

# Roll a die 120 times -- is it fair?
# (We secretly make it slightly loaded)
probs_true <- c(0.12, 0.15, 0.18, 0.20, 0.18, 0.17)
rolls <- sample(1:6, size = 120, replace = TRUE, prob = probs_true)

observed <- table(rolls)
expected <- rep(120 / 6, 6)

# Manual chi-square statistic
chi_sq <- sum((observed - expected)^2 / expected)
df_gof <- length(observed) - 1
p_value_gof <- pchisq(chi_sq, df = df_gof, lower.tail = FALSE)

data.frame(
  chi_sq = round(chi_sq, 4),
  df = df_gof,
  p_value = round(p_value_gof, 4)
)
```


## 11.7 Chi-Square Test of Independence

### 11.7.1 The Problem

We observe a **contingency table** -- counts cross-classified by two categorical variables with $r$ rows and $c$ columns. We wish to test:

$$H_0: \text{the two variables are independent} \quad \text{vs} \quad H_1: \text{they are associated.}$$

### 11.7.2 Derivation

**Step 1.** Under independence, $P(\text{row } i \cap \text{col } j) = P(\text{row } i) \cdot P(\text{col } j)$. We estimate these marginal probabilities by:

$$\hat{p}_{i\cdot} = \frac{O_{i\cdot}}{n}, \quad \hat{p}_{\cdot j} = \frac{O_{\cdot j}}{n}$$

where $O_{i\cdot}$ and $O_{\cdot j}$ are the row and column totals respectively.

**Step 2.** The expected count under $H_0$ for cell $(i, j)$ is:

$$E_{ij} = n \cdot \hat{p}_{i\cdot} \cdot \hat{p}_{\cdot j} = \frac{O_{i\cdot} \cdot O_{\cdot j}}{n}.$$

This formula has a satisfying interpretation: we distribute the total count across cells in proportion to the marginal frequencies.

**Step 3.** The test statistic is the same Pearson form:

$$\boxed{X^2 = \sum_{i=1}^{r} \sum_{j=1}^{c} \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \stackrel{\text{approx}}{\sim} \chi^2_{(r-1)(c-1)} \quad \text{under } H_0.}$$

**Step 4.** The degrees of freedom are $(r-1)(c-1)$. This comes from: the table has $rc$ cells, but we impose $r + c - 1$ constraints (the row and column marginals fix the marginal probabilities, minus the redundant constraint that all marginals sum to $n$). So the free parameters are $rc - (r + c - 1) = (r-1)(c-1)$.

### 11.7.3 Streptomycin TB Trial

The 1948 Medical Research Council streptomycin trial was one of the first randomised controlled trials. Patients with pulmonary tuberculosis were randomly assigned to streptomycin or control. We test whether treatment is associated with outcome.


``` r

    13, 12, 7, 20),  # Control row
  nrow = 2, byrow = TRUE,
```

``` r
# Manual chi-square test of independence
n_total <- sum(strep_mat)
row_totals <- rowSums(strep_mat)
col_totals <- colSums(strep_mat)

# Expected counts under independence
expected_mat <- outer(row_totals, col_totals) / n_total
round(expected_mat, 2)

# Test statistic
chi_sq_ind <- sum((strep_mat - expected_mat)^2 / expected_mat)
df_ind <- (nrow(strep_mat) - 1) * (ncol(strep_mat) - 1)
p_value_ind <- pchisq(chi_sq_ind, df = df_ind, lower.tail = FALSE)

data.frame(
  chi_sq = round(chi_sq_ind, 4),
  df = df_ind,
  p_value = round(p_value_ind, 4)
)
```
