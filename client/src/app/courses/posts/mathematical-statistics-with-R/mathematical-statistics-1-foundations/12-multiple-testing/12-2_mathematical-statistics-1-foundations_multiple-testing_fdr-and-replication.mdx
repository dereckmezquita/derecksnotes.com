# Chapter 12 -- Multiple Testing

## Part 2: False Discovery Rate and Replication

In Part 1 we introduced the FWER -- the probability of even one false positive. Bonferroni and Holm control the FWER, but they can be extremely conservative when $m$ is large. A genomics study testing 20,000 genes at Bonferroni-corrected $\alpha/m = 0.05/20000 = 0.0000025$ needs overwhelming evidence to declare anything significant. Many genuine effects will be missed.

In 1995, Benjamini and Hochberg proposed a more liberal error criterion: instead of controlling the probability of *any* false positive, control the **proportion** of false positives among all rejections. This is the **false discovery rate** (FDR), and it has transformed modern high-dimensional statistics.

---

## 12.3 The False Discovery Rate

### 12.3.1 Setting Up the Counting Table

When we conduct $m$ hypothesis tests, the outcomes can be summarised in a $2 \times 2$ table:

|  | Not rejected | Rejected | Total |
|:--|:-:|:-:|:-:|
| $H_0$ true | $U$ | $V$ | $m_0$ |
| $H_0$ false | $T$ | $S$ | $m_1$ |
| **Total** | $m - R$ | $R$ | $m$ |

- $R$: total number of rejections (observable).
- $V$: number of false positives (Type I errors) among rejections (unobservable).
- $S$: number of true positives among rejections (unobservable).
- $m_0$: number of true nulls (unknown).
- $m_1 = m - m_0$: number of false nulls (unknown).

### 12.3.2 Definition of FDR

The **false discovery proportion** (FDP) is the fraction of rejections that are false positives:

$$\text{FDP} = \frac{V}{R} = \frac{V}{\max(R, 1)},$$

where we define $V/R = 0$ when $R = 0$ (no rejections means no false discoveries).

The **false discovery rate** is the expected FDP:

$$\text{FDR} = E\!\left[\frac{V}{\max(R, 1)}\right].$$

**Interpretation:** If we control FDR at 5%, then among the hypotheses we declare significant, we expect about 5% to be false positives. This is a much less stringent requirement than FWER, which demands that the probability of *any* false positive is below 5%.

### 12.3.3 Relationship Between FWER and FDR

When all nulls are true ($m_0 = m$), any rejection is a false positive, so $V = R$ and FDP $= 1$ whenever $R > 0$. Therefore:

$$\text{FDR} = E\!\left[\frac{V}{R} \cdot \mathbf{1}(R > 0)\right] = 1 \cdot P(R > 0) = P(V \ge 1) = \text{FWER}.$$

Under the global null, FDR and FWER are identical. When some nulls are false, FDR $\le$ FWER, so FDR control is always at least as permissive as FWER control.

---

## 12.4 The Benjamini-Hochberg (BH) Procedure

### 12.4.1 The Algorithm

Given $m$ p-values $p_1, \ldots, p_m$ and a target FDR level $q$:

1. Order the p-values: $p_{(1)} \le p_{(2)} \le \cdots \le p_{(m)}$.
2. Find the largest $k$ such that $p_{(k)} \le \frac{k}{m} \cdot q$.
3. Reject all hypotheses $H_{(1)}, \ldots, H_{(k)}$.

The BH thresholds form a line: $q/m, \; 2q/m, \; 3q/m, \; \ldots, \; q$. We walk up the sorted p-values and find the last one that falls below this line.

### 12.4.2 Intuition

The BH procedure is a "step-up" procedure. The threshold $kq/m$ for the $k$-th smallest p-value reflects that if we reject $k$ hypotheses, we tolerate a fraction $q$ as false discoveries -- about $kq$ false discoveries out of $k$ rejections.

### 12.4.3 Implementation from Scratch


``` r
set.seed(42)
box::use(data.table)
box::use(ggplot2)

#' Benjamini-Hochberg procedure -- implemented from scratch
#' @param p numeric vector of raw p-values
#' @param q target FDR level
#' @return data.table with original p-values, adjusted p-values, and decisions
my_bh <- function(p, q = 0.05) {
  m <- length(p)
  ord <- order(p)
  p_sorted <- p[ord]

  # BH thresholds
  thresholds <- (seq_len(m) / m) * q

  # Find the largest k where p_(k) <= k*q/m
  below <- p_sorted <= thresholds
  if (!any(below)) {
    k_star <- 0
  } else {
    k_star <- max(which(below))
  }

  # Adjusted p-values: p_(k) * m/k, enforced to be non-increasing from right
  adj_sorted <- p_sorted * m / seq_len(m)
  # Walk backwards to enforce monotonicity
  for (i in (m - 1):1) {
    adj_sorted[i] <- min(adj_sorted[i], adj_sorted[i + 1])
  }
  adj_sorted <- pmin(adj_sorted, 1)

  # Map back to original order
  adj_p <- numeric(m)
  adj_p[ord] <- adj_sorted

  data.table$data.table(
    test = seq_len(m),
    p_raw = p,
    p_adj = adj_p,
    reject = adj_p <= q
  )
}
```

## 12.6 The Replication Crisis

### 12.6.1 What Went Wrong?

Beginning around 2011, researchers discovered that a large fraction of published findings could not be replicated. Several factors contributed:

1. **p-hacking:** Trying multiple analyses and reporting only the one that produced $p < 0.05$ -- the multiple testing problem in disguise.
2. **Publication bias:** Journals published significant results and rejected null results, enriching the literature with false positives.
3. **Low statistical power:** Small studies that find significance tend to overestimate true effect sizes dramatically.
4. **HARKing** (Hypothesising After Results are Known): Presenting exploratory findings as though they were predicted in advance.

### 12.6.2 Simulation: The Winner's Curse


``` r
set.seed(42)

n_studies <- 1000
true_effect <- 0.2   # small true effect (Cohen's d)
n_per_study <- 30    # small sample size (typical of many fields)

# Simulate each study: two-sample t-test
sim_studies <- data.table$data.table(
  study = seq_len(n_studies),
  observed_d = vapply(seq_len(n_studies), function(i) {
    grp1 <- rnorm(n_per_study, mean = 0,           sd = 1)
    grp2 <- rnorm(n_per_study, mean = true_effect,  sd = 1)
    tt <- t.test(grp1, grp2)
```

### 12.6.4 Pre-Registration and the Solution

**Pre-registration** addresses the root cause by requiring researchers to specify hypotheses, sample sizes, and analyses *before* collecting data. Key elements: (1) primary hypothesis stated precisely, (2) sample size justified by power calculation, (3) exact statistical test and stopping rules, (4) multiple testing correction specified in advance, (5) clear distinction between confirmatory and exploratory analyses.

### 12.6.5 Effect Sizes vs p-Values

A p-value tells us whether an effect is *statistically distinguishable from zero* but says nothing about its *magnitude*. With a large enough sample, even a trivially small effect becomes significant.

## 12.7 Practical Guidelines: When to Use What

### 12.7.1 Decision Framework

| Scenario | Recommended method | Rationale |
|:---------|:-------------------|:----------|
| A few planned comparisons ($m < 10$) | Holm | FWER control; more powerful than Bonferroni |
| Confirmatory clinical trial with co-primary endpoints | Bonferroni or Holm | Regulatory requirement for strong FWER control |
| Exploratory genomics / proteomics ($m > 1000$) | BH | FWER too strict; FDR balances discovery and error |
| Single pre-registered primary analysis | No correction needed | Only one test; $\alpha$ is already the error rate |
| Exploratory subgroup analysis | BH or report as exploratory | Flag findings as hypothesis-generating |

### 12.7.2 Rules of Thumb

1. **Always report** the number of tests and the correction method.
2. **Decide before testing:** choose the correction *before* seeing data.
3. **When in doubt, use Holm:** at least as powerful as Bonferroni with FWER control.
4. **When FWER is too strict, use BH:** far more powerful for large $m$.
5. **Report effect sizes** alongside p-values.
6. **Pre-register** confirmatory analyses.

### 12.7.3 Power Simulation: Comparing Methods at Scale

``` r
set.seed(42)

sim_power <- function(m, m1, effect, n_sims = 500) {
  m0 <- m - m1
  powers <- vapply(seq_len(n_sims), function(i) {
    z_null <- rnorm(m0)

      bonf = mean(p.adjust(p_all, "bonferroni")[truly_alt] <= 0.05),
      holm = mean(p.adjust(p_all, "holm")[truly_alt] <= 0.05),
```

## 12.8 FDR Simulation Verification

To confirm that the BH procedure truly controls FDR, we run a large-scale simulation.


``` r
set.seed(42)
n_sims <- 5000
m <- 100
m1 <- 15
effect <- 3
q <- 0.05

fdp_values <- vapply(seq_len(n_sims), function(i) {
  z_null <- rnorm(m - m1)
  z_alt  <- rnorm(m1, mean = effect)
  p_all  <- 2 * pnorm(-abs(c(z_null, z_alt)))
  truly_null <- c(rep(TRUE, m - m1), rep(FALSE, m1))

  adj_p <- p.adjust(p_all, method = "BH")
  rejected <- adj_p <= q
  R <- sum(rejected)
  V <- sum(rejected & truly_null)

  if (R == 0) 0 else V / R
}, numeric(1))

data.table$data.table(
  quantity = c("Mean FDP (= FDR)", "Median FDP",
               "SD of FDP", "Target q"),
  value = round(c(mean(fdp_values), median(fdp_values),
                  sd(fdp_values), q), 4)
)
```

## Communicating to Stakeholders

When explaining FDR and the replication crisis to non-technical audiences:

- Use an analogy: "FWER requires zero defectives; FDR tolerates a 5% defect rate. FDR is more practical when screening thousands of items."
- Explain the replication crisis: "Many findings turned out to be false alarms because researchers tested many things but only reported the significant ones."
- Explain pre-registration: "It is like submitting your exam answers before seeing the questions."
- When reporting FDR results: "Among 150 significant genes, we expect about 5% (roughly 7 or 8) to be false positives."
- Clarify: "FDR tells us about the overall rate, not about any individual finding."

## Quick Reference

| Concept | Formula / Rule | Notes |
|:--------|:---------------|:------|
| **FDP** | $V / \max(R, 1)$ | Proportion of false discoveries; random |
| **FDR** | $E[\text{FDP}]$ | Expected false discovery proportion |
| **FDR = FWER** | Under the global null | When all nulls are true |
| **BH procedure** | Reject $H_{(1)}, \ldots, H_{(k^*)}$ where $k^*$ is largest $k$ with $p_{(k)} \le kq/m$ | Controls FDR at level $q$ under independence |
| **BH adjusted p-value** | $\tilde{p}_{(k)} = \min\bigl(\frac{m}{k} p_{(k)}, \; 1\bigr)$ (enforced non-decreasing) | Compare to $q$ directly |
| **Winner's curse** | Significant effects overestimate truth | Especially severe in underpowered studies |
| **R function** | `p.adjust(p, method = "BH")` | Also `"fdr"` is an alias |

**Key insight:** FDR control tolerates a small fraction of false discoveries in exchange for much greater power to detect true signals. Use it when discovery matters more than certainty about each individual finding.

## Exercises

### Derivation Exercises

**Exercise 12.10.** Prove that under the global null (all $m$ hypotheses are true), $\text{FDR} = \text{FWER}$. *(Hint: when all nulls are true, $V = R$, so FDP = 1 whenever $R > 0$ and FDP = 0 when $R = 0$.)*

**Exercise 12.11.** Show that if the BH procedure is applied with $q = \alpha/m$, it reduces to the Bonferroni correction. *(Hint: the BH threshold for the $k$-th p-value becomes $k \cdot \alpha/(m \cdot m)$.)*

**Exercise 12.12.** Prove that the expected number of false discoveries under BH is at most $m_0 \cdot q / m$, where $m_0$ is the number of true nulls. *(Hint: this follows from the BH theorem: $\text{FDR} \le m_0 q / m \le q$.)*

### Computation Exercises

**Exercise 12.13.** A gene expression study tests 500 genes. Of these, 480 are true nulls and 20 have real effects. Using the BH procedure at $q = 0.10$, what is the maximum number of false discoveries we "budget" for? If we reject 35 genes, what is our estimated FDP?

**Exercise 12.14.** A researcher reports: "We applied the BH correction at FDR = 0.05 and found 200 significant genes out of 10,000 tested." Approximately how many of these 200 genes do we expect to be false discoveries?

### R Exercises

**Exercise 12.15.** Implement a simulation comparing the actual FDR of BH across proportions of true nulls: $\pi_0 = 0.5, 0.7, 0.9, 0.95, 1.0$. For each $\pi_0$, simulate 2000 experiments with $m = 100$ tests, compute the average FDP, and confirm FDR $\le q$.

**Exercise 12.16.** Using the NHANES data from Part 1, apply BH to the 20 health outcome tests. Compare the number of rejections to Bonferroni and Holm.

### Spot-the-Error Exercises

**Exercise 12.17.** A researcher uses the BH procedure at $q = 0.05$, finds 40 significant results, and claims: "Each of these 40 findings has at most a 5% chance of being a false positive." What is wrong with this interpretation?

**Exercise 12.18.** A journal reviewer states: "You tested 10,000 genes and found only 50 significant after BH correction. Clearly your experiment failed -- you should have used an uncorrected threshold to find more results." How would you respond?

**Exercise 12.19.** A student runs 200 t-tests, applies BH at $q = 0.05$, and obtains 0 rejections. They then re-run the analysis with $q = 0.20$ and find 15 rejections. They report both analyses, choosing to emphasise the second. What statistical and ethical issues arise?
