# Chapter 15 -- Simple Linear Regression

## Part 3: R-squared and Residuals

We have derived the OLS estimators and their sampling distributions. But how good is the fit? In this part we derive $R^2$ and adjusted $R^2$, connect them to the $F$-test, and then examine residual diagnostics -- the essential practice of checking whether the model assumptions are met. We close with three real-world datasets that illustrate different regression stories.

---

## 15.18 The Coefficient of Determination $R^2$

### 15.18.1 Definition and Derivation

Recall the decomposition from Part 1:

$$\text{SST} = \text{SSReg} + \text{SSE}$$

The **coefficient of determination** is:

$$R^2 = \frac{\text{SSReg}}{\text{SST}} = 1 - \frac{\text{SSE}}{\text{SST}}$$

$R^2$ measures the **proportion of variance in $Y$ that is explained by the linear relationship with $X$**. It satisfies $0 \le R^2 \le 1$.

### 15.18.2 $R^2$ Equals the Squared Correlation

**Theorem.** In simple linear regression, $R^2 = r_{xy}^2$, where $r_{xy}$ is the Pearson correlation coefficient.

*Proof.* We have:

$$r_{xy} = \frac{S_{xy}}{\sqrt{S_{xx} \cdot S_{yy}}} = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2 \cdot \sum(y_i - \bar{y})^2}}$$

and $\hat{\beta}_1 = S_{xy}/S_{xx}$. Now:

$$\text{SSReg} = \sum(\hat{y}_i - \bar{y})^2 = \hat{\beta}_1^2 \sum(x_i - \bar{x})^2 = \hat{\beta}_1^2 S_{xx} = \frac{S_{xy}^2}{S_{xx}}$$

Therefore:

$$R^2 = \frac{\text{SSReg}}{\text{SST}} = \frac{S_{xy}^2}{S_{xx} \cdot S_{yy}} = r_{xy}^2 \quad \square$$

This is a beautiful result: the proportion of variance explained is exactly the square of the correlation. A correlation of $r = 0.8$ means $R^2 = 0.64$, so 64% of the variation in $Y$ is explained by $X$.

### 15.18.3 Limitations of $R^2$

$R^2$ is seductive but has important limitations:

1. **$R^2$ always increases** when you add more predictors (even useless ones). This is a problem in multiple regression.
2. **A high $R^2$ does not imply the model is correct.** Anscombe's quartet shows four datasets with identical $R^2$ but wildly different patterns.
3. **$R^2$ measures linear association only.** A perfect quadratic relationship will have $R^2 < 1$ in a linear model.
4. **$R^2$ depends on the variance of $X$.** If you restrict the range of $X$, $R^2$ drops even if the true relationship is unchanged.

### 15.18.4 Computing $R^2$ from Scratch


``` r
data_dir <- "../../../data"
cats <- data.table$fread(file.path(data_dir, "classic/cats.csv"))

  value = c(R2, r_xy^2, summary(fit)$r.squared)
)
```

## 15.19 Adjusted $R^2$

### 15.19.1 The Problem

Since $R^2$ never decreases when predictors are added, it is a biased measure of model quality. We need a version that penalises model complexity.

### 15.19.2 Derivation

The adjusted $R^2$ is:

$$R^2_{\text{adj}} = 1 - \frac{\text{SSE}/(n - p)}{\text{SST}/(n - 1)} = 1 - \frac{n - 1}{n - p}(1 - R^2)$$

where $p$ is the total number of parameters (including the intercept). For simple regression, $p = 2$.

The idea: we replace raw sums of squares with **mean squares** (i.e., divide by degrees of freedom). The mean squared error $\text{MSE} = \text{SSE}/(n-p)$ is an unbiased estimator of $\sigma^2$, so $R^2_{\text{adj}}$ is roughly the proportion of variance explained after correcting for the number of parameters.


``` r
n <- nrow(cats)
p <- 2  # intercept + slope
R2_adj <- 1 - (n - 1) / (n - p) * (1 - R2)

data.frame(
  quantity = c("R^2", "Adjusted R^2", "From summary()"),
  value = c(R2, R2_adj, summary(fit)$adj.r.squared)
)
```

```
        quantity     value
1            R^2 0.6466209
2   Adjusted R^2 0.6441323
3 From summary() 0.6441323
```

Note that $R^2_{\text{adj}} < R^2$. In simple regression the difference is small, but in multiple regression with many predictors, the gap can be substantial.

## 15.20 Connection Between $R^2$ and the $F$-test

### 15.20.1 The Relationship

The $F$-statistic for testing $H_0: \beta_1 = 0$ in simple regression is:

$$F = \frac{\text{SSReg}/1}{\text{SSE}/(n-2)} = \frac{\text{MSReg}}{\text{MSE}}$$

We can express this purely in terms of $R^2$:

$$F = \frac{R^2 / 1}{(1 - R^2)/(n - 2)} = \frac{R^2(n - 2)}{1 - R^2}$$

This shows that the $F$-test and $R^2$ are equivalent: a large $R^2$ corresponds to a large $F$-statistic, which corresponds to a small $p$-value.


``` r
F_from_R2 <- R2 * (n - 2) / (1 - R2)
F_from_anova <- summary(fit)$fstatistic[1]

data.frame(
  method = c("from R^2", "from summary()"),
  F_stat = c(F_from_R2, F_from_anova)
)
```

```
              method   F_stat
            from R^2 259.8348
value from summary() 259.8348
```

## 15.21 Residual Analysis

### 15.21.1 Why Check Residuals?

All the inference in Parts 1 and 2 depends on the assumptions: linearity, independence, homoscedasticity, and normality. If these fail, confidence intervals and $p$-values may be unreliable. **Residual analysis** is our primary tool for checking these assumptions.

### 15.21.2 Standardised and Studentised Residuals

Raw residuals $e_i = y_i - \hat{y}_i$ do not all have the same variance. In fact:

$$\text{Var}(e_i) = \sigma^2(1 - h_{ii})$$

where $h_{ii}$ is the $i$th diagonal element of the **hat matrix** $H = X(X'X)^{-1}X'$. The **internally studentised residuals** are:

$$r_i = \frac{e_i}{s\sqrt{1 - h_{ii}}}$$

and the **externally studentised (deleted) residuals** use a leave-one-out estimate $s_{(i)}$:

$$t_i = \frac{e_i}{s_{(i)}\sqrt{1 - h_{ii}}} \sim t_{n-p-1}$$

**Plot 1 -- Residuals vs Fitted**: the residuals should scatter randomly around zero with constant spread. A funnel shape indicates heteroscedasticity; a curve indicates non-linearity.

**Plot 2 -- Normal Q-Q**: the standardised residuals should fall along the diagonal line. Deviations in the tails suggest heavy or light tails.

**Plot 3 -- Scale-Location**: $\sqrt{|\text{standardised residuals}|}$ vs fitted values. A horizontal line indicates constant variance.

**Plot 4 -- Residuals vs Leverage**: identifies influential points. Points with high leverage and large residuals have outsized influence on the fit.

## 15.25 Formal Tests for Assumptions

### 15.25.1 Shapiro-Wilk Test for Normality


``` r
shapiro.test(residuals(fit))
```

```

	Shapiro-Wilk normality test

data:  residuals(fit)
W = 0.9845, p-value = 0.1046
```

The $p$-value is small, suggesting the residuals are not perfectly normal. However, with $n = 144$ observations, the $t$-test is robust to mild non-normality due to the Central Limit Theorem.

### 15.25.2 Breusch-Pagan-style Test for Heteroscedasticity

We can test for non-constant variance by regressing the squared residuals on $\hat{y}$:


``` r
# Simple Breusch-Pagan from scratch
e2 <- residuals(fit)^2
bp_fit <- lm(e2 ~ fitted(fit))
bp_stat <- nrow(cats) * summary(bp_fit)$r.squared

c(
  "BP statistic" = bp_stat,
  "p-value" = pchisq(bp_stat, df = 1, lower.tail = FALSE)
)
```

```
BP statistic      p-value 
 9.529395820  0.002022067 
```

### 15.25.3 Durbin-Watson-style Test for Autocorrelation

If the data have a natural ordering (e.g., time series), we check for autocorrelation in the residuals:


``` r
# Durbin-Watson from scratch
e <- residuals(fit)
n <- length(e)
DW <- sum(diff(e)^2) / sum(e^2)

c("DW statistic" = DW)
```

```
DW statistic 
    1.579896 
```

A value near 2 indicates no autocorrelation. Values substantially below 2 suggest positive autocorrelation; values substantially above 2 suggest negative autocorrelation. For the cats data (which have no natural ordering), this test is not particularly meaningful.

## 15.26 When Simple Linear Regression Fails

Simple linear regression fails when:

1. **The true relationship is non-linear** -- use transformations or polynomial regression.
2. **There are influential outliers** -- use robust regression or remove outliers after careful investigation.
3. **The errors are heteroscedastic** -- use weighted least squares or variance-stabilising transformations.
4. **Important predictors are omitted** -- move to multiple regression (Chapter 16).
5. **The response is bounded** (e.g., proportions, counts) -- use generalised linear models (Chapters 19-21).

## Communicating to Stakeholders

When reporting $R^2$ and diagnostics:

- **Do not over-sell $R^2$**. An $R^2$ of 0.65 does not mean the model is 65% "correct" -- it means 65% of the variation is explained. The remaining 35% is unexplained.
- **Show residual plots alongside your results.** A model with $R^2 = 0.95$ is worthless if the residual plot reveals a systematic curve.
- **Simpson's paradox**: always ask "could this relationship reverse when we account for subgroups?" Present stratified analyses alongside the overall analysis.
- **Use language like** "after accounting for body weight, we find..." rather than "body weight causes..."
