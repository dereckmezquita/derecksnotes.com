# Chapter 15 -- Simple Linear Regression

## Part 2: Inference and Prediction

In Part 1 we derived the OLS estimators $\hat{\beta}_0$ and $\hat{\beta}_1$ as the values that minimise the sum of squared residuals. But point estimates alone tell us nothing about uncertainty. In this part we develop the full inferential machinery: we prove that the estimators are unbiased, derive their sampling distributions, construct confidence intervals and hypothesis tests, and carefully distinguish between **confidence intervals** for the mean response and **prediction intervals** for a new observation.

---

## 15.10 Properties of the OLS Estimators

### 15.10.1 Unbiasedness of $\hat{\beta}_1$

**Theorem.** Under the Gauss-Markov conditions, $E[\hat{\beta}_1] = \beta_1$.

*Proof.* Recall:

$$\hat{\beta}_1 = \frac{\sum(x_i - \bar{x})(Y_i - \bar{Y})}{\sum(x_i - \bar{x})^2} = \frac{\sum(x_i - \bar{x})Y_i}{S_{xx}}$$

where the last step uses $\sum(x_i - \bar{x})\bar{Y} = \bar{Y}\sum(x_i - \bar{x}) = 0$.

Define $c_i = (x_i - \bar{x})/S_{xx}$. Then $\hat{\beta}_1 = \sum c_i Y_i$ -- the slope estimator is a **linear function** of the $Y_i$'s. Note that $\sum c_i = 0$ and $\sum c_i x_i = 1$.

Taking expectations:

$$E[\hat{\beta}_1] = \sum c_i E[Y_i] = \sum c_i(\beta_0 + \beta_1 x_i) = \beta_0\sum c_i + \beta_1\sum c_i x_i = 0 + \beta_1 \cdot 1 = \beta_1 \quad \square$$

### 15.10.2 Unbiasedness of $\hat{\beta}_0$

**Theorem.** Under the Gauss-Markov conditions, $E[\hat{\beta}_0] = \beta_0$.

*Proof.* $\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1\bar{x}$, so:

$$E[\hat{\beta}_0] = E[\bar{Y}] - \bar{x}\,E[\hat{\beta}_1] = (\beta_0 + \beta_1\bar{x}) - \bar{x}\beta_1 = \beta_0 \quad \square$$

### 15.10.3 Variance of $\hat{\beta}_1$

**Theorem.** $\text{Var}(\hat{\beta}_1) = \sigma^2/S_{xx}$.

*Proof.* Since $\hat{\beta}_1 = \sum c_i Y_i$ and the $Y_i$'s are independent with $\text{Var}(Y_i) = \sigma^2$:

$$\text{Var}(\hat{\beta}_1) = \sum c_i^2 \,\text{Var}(Y_i) = \sigma^2 \sum c_i^2 = \sigma^2 \cdot \frac{\sum(x_i - \bar{x})^2}{S_{xx}^2} = \frac{\sigma^2}{S_{xx}} \quad \square$$

This tells us the slope is estimated more precisely when:
- The error variance $\sigma^2$ is small (less noise).
- $S_{xx} = \sum(x_i - \bar{x})^2$ is large (the $x$-values are more spread out).

### 15.10.4 Variance of $\hat{\beta}_0$

**Theorem.** $\text{Var}(\hat{\beta}_0) = \sigma^2\left(\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right)$.

*Proof.* $\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1\bar{x}$. We can write $\hat{\beta}_0 = \sum d_i Y_i$ where $d_i = 1/n - \bar{x}\,c_i$. Then:

$$\text{Var}(\hat{\beta}_0) = \sigma^2\sum d_i^2 = \sigma^2\left(\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right) \quad \square$$

Note that the intercept is estimated more precisely when $\bar{x}$ is close to zero.

### 15.10.5 Covariance of $\hat{\beta}_0$ and $\hat{\beta}_1$

$$\text{Cov}(\hat{\beta}_0, \hat{\beta}_1) = -\sigma^2\frac{\bar{x}}{S_{xx}}$$

The estimators are negatively correlated when $\bar{x} > 0$: if we overestimate the slope, we tend to underestimate the intercept, and vice versa.

### 15.10.6 BLUE Property (Preview)

The Gauss-Markov theorem (which we prove fully in Chapter 16) guarantees that among all linear unbiased estimators of $\beta_1$, the OLS estimator has the **smallest variance**. OLS is the **Best Linear Unbiased Estimator** (BLUE). For the simple regression case, the proof reduces to showing that for any constants $a_i$ with $\sum a_i = 0$ and $\sum a_i x_i = 1$:

$$\text{Var}\!\left(\sum a_i Y_i\right) = \sigma^2\sum a_i^2 \ge \sigma^2\sum c_i^2 = \frac{\sigma^2}{S_{xx}}$$

with equality if and only if $a_i = c_i$. This is Cauchy-Schwarz.

---

## 15.11 Sampling Distributions Under Normality

If we additionally assume $\varepsilon_i \sim N(0, \sigma^2)$, then:

$$\hat{\beta}_1 \sim N\!\left(\beta_1,\, \frac{\sigma^2}{S_{xx}}\right)$$

$$\hat{\beta}_0 \sim N\!\left(\beta_0,\, \sigma^2\!\left(\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right)\right)$$

Moreover, $\hat{\beta}_1$ and $s^2 = \text{SSE}/(n-2)$ are independent, and:

$$\frac{(n-2)s^2}{\sigma^2} \sim \chi^2_{n-2}$$

Therefore the pivotal quantity:

$$T = \frac{\hat{\beta}_1 - \beta_1}{s/\sqrt{S_{xx}}} = \frac{\hat{\beta}_1 - \beta_1}{\text{SE}(\hat{\beta}_1)} \sim t_{n-2}$$

follows a $t$-distribution with $n - 2$ degrees of freedom.

---

## 15.12 Hypothesis Tests for the Slope

### 15.12.1 The Most Important Test

The most common test in simple regression is:

$$H_0: \beta_1 = 0 \quad \text{vs} \quad H_1: \beta_1 \ne 0$$

Under $H_0$, $X$ has no linear relationship with $Y$. The test statistic is:

$$t = \frac{\hat{\beta}_1}{\text{SE}(\hat{\beta}_1)} = \frac{\hat{\beta}_1}{s/\sqrt{S_{xx}}}$$

We reject $H_0$ at level $\alpha$ if $|t| > t_{n-2, \alpha/2}$.

## 15.13 Confidence Intervals for $\beta_0$ and $\beta_1$

A $(1-\alpha)$ confidence interval for $\beta_1$ is:

$$\hat{\beta}_1 \pm t_{n-2, \alpha/2} \cdot \text{SE}(\hat{\beta}_1)$$


``` r
# 95% CI from scratch
alpha <- 0.05
n <- nrow(cats)
Sxx <- sum((cats$Bwt - mean(cats$Bwt))^2)
se_b1 <- s$sigma / sqrt(Sxx)
t_crit <- qt(1 - alpha / 2, df = n - 2)

ci_scratch <- coef(fit)[2] + c(-1, 1) * t_crit * se_b1

# Compare with confint()
ci_builtin <- confint(fit, level = 0.95)

data.frame(
  method = c("from scratch", "confint()"),
  lower = c(ci_scratch[1], ci_builtin[2, 1]),
  upper = c(ci_scratch[2], ci_builtin[2, 2])
)
```


**Interpretation**: we are 95% confident that each additional kilogram of body weight is associated with between 3.54 and 4.53 additional grams of heart weight.

## 15.14 Confidence Intervals vs Prediction Intervals

This distinction is one of the most commonly confused topics in applied statistics. Both intervals are centred at the same fitted value, but they answer **different questions**.

### 15.14.1 Confidence Interval for the Mean Response

*Question*: what is the **average** heart weight of all cats with body weight $x_0$?

We are estimating $E[Y \mid X = x_0] = \beta_0 + \beta_1 x_0$. The point estimate is $\hat{y}_0 = \hat{\beta}_0 + \hat{\beta}_1 x_0$, and its variance is:

$$\text{Var}(\hat{y}_0) = \sigma^2\left(\frac{1}{n} + \frac{(x_0 - \bar{x})^2}{S_{xx}}\right)$$

The $(1-\alpha)$ confidence interval for the mean response is:

$$\hat{y}_0 \pm t_{n-2, \alpha/2} \cdot s\sqrt{\frac{1}{n} + \frac{(x_0 - \bar{x})^2}{S_{xx}}}$$

### 15.14.2 Prediction Interval for a New Observation

*Question*: what heart weight will the **next individual cat** with body weight $x_0$ have?

Now we are predicting an individual $Y_{\text{new}} = \beta_0 + \beta_1 x_0 + \varepsilon_{\text{new}}$. The prediction error is:

$$Y_{\text{new}} - \hat{y}_0 = (\beta_0 + \beta_1 x_0 + \varepsilon_{\text{new}}) - \hat{y}_0$$

and its variance is:

$$\text{Var}(Y_{\text{new}} - \hat{y}_0) = \sigma^2\left(1 + \frac{1}{n} + \frac{(x_0 - \bar{x})^2}{S_{xx}}\right)$$

The extra "$1$" accounts for the inherent variability $\varepsilon_{\text{new}}$ of the new observation itself. The $(1-\alpha)$ prediction interval is:

$$\hat{y}_0 \pm t_{n-2, \alpha/2} \cdot s\sqrt{1 + \frac{1}{n} + \frac{(x_0 - \bar{x})^2}{S_{xx}}}$$

### 15.14.3 The Key Difference

- The **confidence interval** (CI) shrinks to zero as $n \to \infty$ -- with enough data, we can nail down the mean response exactly.
- The **prediction interval** (PI) never shrinks below $\pm t \cdot s$ -- even with infinite data, individual observations still vary by $\pm \sigma$.

### 15.14.5 Implementation from Scratch


``` r
#' Compute confidence and prediction intervals from scratch
#' @param fit_lm an lm object
#' @param x0 predictor value(s) at which to predict
#' @param alpha significance level
#' @return data.table with fitted value and both intervals
my_intervals <- function(fit_lm, x0, alpha = 0.05) {
  b <- coef(fit_lm)
  y_hat <- b[1] + b[2] * x0

  x <- fit_lm$model[, 2]
  n <- length(x)
  x_bar <- mean(x)
  Sxx <- sum((x - x_bar)^2)
  s <- summary(fit_lm)$sigma
  t_crit <- qt(1 - alpha / 2, df = n - 2)

  se_ci <- s * sqrt(1/n + (x0 - x_bar)^2 / Sxx)
  se_pi <- s * sqrt(1 + 1/n + (x0 - x_bar)^2 / Sxx)

  data.table$data.table(
    x0 = x0,
    fitted = y_hat,
    ci_lower = y_hat - t_crit * se_ci,
    ci_upper = y_hat + t_crit * se_ci,
    pi_lower = y_hat - t_crit * se_pi,
    pi_upper = y_hat + t_crit * se_pi
  )
}

# Predict at x0 = 2.5, 3.0, 3.5 kg
my_intervals(fit, x0 = c(2.5, 3.0, 3.5))
```

## 15.15 The Danger of Extrapolation

Regression models are valid only within (or near) the range of the observed data. Predicting far beyond the observed $x$-range is called **extrapolation** and can produce wildly misleading results.

## 15.16 Simulation: Verifying the Coverage

A confidence interval is only useful if it actually contains the true parameter with the claimed probability. Let us run a simulation to verify this.


``` r
set.seed(42)
n_sims <- 200
n <- 50
beta0_true <- 2
beta1_true <- 3
sigma_true <- 4

results <- data.table$data.table(
  sim = seq_len(n_sims),
  b1_hat = numeric(n_sims),
  ci_lower = numeric(n_sims),
  ci_upper = numeric(n_sims)
)

for (k in seq_len(n_sims)) {
  x <- runif(n, 0, 10)
  y <- beta0_true + beta1_true * x + rnorm(n, 0, sigma_true)
  fit_sim <- lm(y ~ x)
  ci <- confint(fit_sim)["x", ]
  results[k, `:=`(
    b1_hat = coef(fit_sim)[2],
    ci_lower = ci[1],
    ci_upper = ci[2]
  )]
}

results[, covers := ci_lower <= beta1_true & beta1_true <= ci_upper]

coverage <- mean(results$covers)
coverage

# Plot first 50 intervals
plot_sub <- results[1:50]
plot_sub[, sim := .I]

ggplot2$ggplot(plot_sub, ggplot2$aes(
  x = b1_hat, y = sim,
  xmin = ci_lower, xmax = ci_upper,
  colour = covers
)) +
  ggplot2$geom_errorbarh(Height = 0.3) +
  ggplot2$geom_point(size = 1) +
  ggplot2$geom_vline(xintercept = beta1_true, linetype = "dashed") +
  ggplot2$scale_colour_manual(values = c("TRUE" = "#0072B2", "FALSE" = "#D55E00")) +
  ggplot2$labs(
    x = expression(hat(beta)[1]),
    y = "Simulation",
    title = sprintf("95%% CIs for beta_1 (coverage = %.1f%%)", 100 * coverage)
  ) +
  ggplot2$theme_minimal(base_size = 13) +
  ggplot2$theme(legend.position = "none")
```

## Communicating to Stakeholders

When presenting regression inference:

- **Report confidence intervals, not just $p$-values.** A stakeholder needs to know the plausible range of the effect, not just whether it is "significant."
- **Distinguish prediction from estimation.** If a manager asks "what will the next cat weigh?", they want a **prediction interval**. If they ask "what is the average heart weight for 3 kg cats?", they want a **confidence interval**.
- **Explain the width of the interval.** "Our best guess is 12.5 g, but the actual value for any individual cat could plausibly be anywhere between 6 g and 19 g."
- **Never extrapolate without warning.** If the data covers body weights from 2 to 4 kg, explicitly flag that predictions at 1 kg or 5 kg are unreliable.

## Quick Reference

| Quantity | Formula |
|:---------|:--------|
| $\text{Var}(\hat{\beta}_1)$ | $\sigma^2 / S_{xx}$ |
| $\text{Var}(\hat{\beta}_0)$ | $\sigma^2(1/n + \bar{x}^2/S_{xx})$ |
| $\text{SE}(\hat{\beta}_1)$ | $s/\sqrt{S_{xx}}$ |
| **$t$-statistic** | $\hat{\beta}_1/\text{SE}(\hat{\beta}_1) \sim t_{n-2}$ |
| **CI for mean** | $\hat{y}_0 \pm t_{n-2,\alpha/2}\cdot s\sqrt{1/n + (x_0-\bar{x})^2/S_{xx}}$ |
| **PI for new obs** | $\hat{y}_0 \pm t_{n-2,\alpha/2}\cdot s\sqrt{1 + 1/n + (x_0-\bar{x})^2/S_{xx}}$ |
| **Key insight** | PI never shrinks below $\pm t\cdot s$; CI $\to 0$ as $n\to\infty$ |
