# Chapter 15 -- Simple Linear Regression

## Part 1: Model and Estimation

We have now assembled every tool we need: probability, random variables, distributions, estimation, hypothesis testing, and linear algebra. It is time to put them all to work on the most important modelling technique in applied statistics -- **linear regression**.

The central question of regression is deceptively simple: given observations of two variables $X$ and $Y$, can we describe how $Y$ changes as $X$ changes? In this first part we formalise the model, derive the least-squares estimators from scratch using calculus, and implement them in R without relying on any built-in fitting function.

---

## 15.1 The Simple Linear Regression Model

### 15.1.1 Setup and Assumptions

Suppose we observe $n$ pairs $(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)$. We model the relationship as:

$$Y_i = \beta_0 + \beta_1 x_i + \varepsilon_i, \quad i = 1, 2, \ldots, n$$

where:

- $\beta_0$ is the **intercept** (the expected value of $Y$ when $X = 0$).
- $\beta_1$ is the **slope** (the expected change in $Y$ per unit increase in $X$).
- $\varepsilon_i$ is the **random error** for observation $i$.

We assume:

1. **Linearity**: the conditional mean $E[Y_i \mid x_i] = \beta_0 + \beta_1 x_i$ is a linear function of $x_i$.
2. **Independence**: the errors $\varepsilon_1, \ldots, \varepsilon_n$ are independent.
3. **Homoscedasticity**: $\text{Var}(\varepsilon_i) = \sigma^2$ for all $i$ (constant variance).
4. **Zero mean errors**: $E[\varepsilon_i] = 0$ for all $i$.

Under these four assumptions, we say the model satisfies the **Gauss-Markov conditions**. When we additionally assume $\varepsilon_i \sim N(0, \sigma^2)$, we have the **normal linear model** and can perform exact inference.

### 15.1.2 What the Model Means Geometrically

The equation $E[Y \mid X = x] = \beta_0 + \beta_1 x$ describes a line in the $(x, y)$-plane. Each observed point $(x_i, y_i)$ lies some vertical distance above or below this line. That vertical distance is the residual $\varepsilon_i$. Our task is to find the line that, in some precise sense, makes these vertical distances as small as possible.

## 15.2 The Least Squares Criterion

### 15.2.1 Defining the Sum of Squared Residuals

Given candidate values $b_0$ and $b_1$ for the intercept and slope, the **predicted value** for observation $i$ is:

$$\hat{y}_i = b_0 + b_1 x_i$$

The **residual** is:

$$e_i = y_i - \hat{y}_i = y_i - b_0 - b_1 x_i$$

The **sum of squared residuals** (SSR) is:

$$S(b_0, b_1) = \sum_{i=1}^{n} e_i^2 = \sum_{i=1}^{n} (y_i - b_0 - b_1 x_i)^2$$

The **Ordinary Least Squares** (OLS) estimators $\hat{\beta}_0$ and $\hat{\beta}_1$ are the values that minimise $S(b_0, b_1)$.

### 15.2.2 Why Squares?

We square the residuals for several reasons:

- Squaring prevents positive and negative residuals from cancelling.
- The squared function is differentiable everywhere, which makes optimisation tractable.
- Under normality of errors, minimising SSR is equivalent to maximising the likelihood (we prove this below).

### 15.2.3 Connection to Maximum Likelihood

If $\varepsilon_i \sim N(0, \sigma^2)$, then $Y_i \sim N(\beta_0 + \beta_1 x_i, \sigma^2)$. The likelihood is:

$$L(\beta_0, \beta_1, \sigma^2) = \prod_{i=1}^{n} \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left(-\frac{(y_i - \beta_0 - \beta_1 x_i)^2}{2\sigma^2}\right)$$

The log-likelihood is:

$$\ell = -\frac{n}{2}\log(2\pi) - \frac{n}{2}\log(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(y_i - \beta_0 - \beta_1 x_i)^2$$

Maximising $\ell$ with respect to $\beta_0$ and $\beta_1$ (for any fixed $\sigma^2 > 0$) is equivalent to minimising $\sum(y_i - \beta_0 - \beta_1 x_i)^2$. Thus OLS and MLE yield the same point estimates for the regression coefficients.

---

## 15.3 Deriving the OLS Estimators

### 15.3.1 Taking Partial Derivatives

We minimise $S(b_0, b_1) = \sum_{i=1}^{n} (y_i - b_0 - b_1 x_i)^2$ by setting the partial derivatives to zero.

**Partial derivative with respect to $b_0$:**

$$\frac{\partial S}{\partial b_0} = -2\sum_{i=1}^{n}(y_i - b_0 - b_1 x_i) = 0$$

Dividing by $-2$ and expanding:

$$\sum_{i=1}^{n} y_i - nb_0 - b_1\sum_{i=1}^{n} x_i = 0$$

$$\Rightarrow \quad b_0 = \bar{y} - b_1\bar{x} \tag{1}$$

**Partial derivative with respect to $b_1$:**

$$\frac{\partial S}{\partial b_1} = -2\sum_{i=1}^{n} x_i(y_i - b_0 - b_1 x_i) = 0$$

Expanding:

$$\sum_{i=1}^{n} x_i y_i - b_0\sum_{i=1}^{n} x_i - b_1\sum_{i=1}^{n} x_i^2 = 0 \tag{2}$$

### 15.3.2 Solving the Normal Equations

Substituting equation (1) into equation (2):

$$\sum x_i y_i - (\bar{y} - b_1\bar{x})\sum x_i - b_1\sum x_i^2 = 0$$

$$\sum x_i y_i - \bar{y}\sum x_i + b_1\bar{x}\sum x_i - b_1\sum x_i^2 = 0$$

$$\sum x_i y_i - n\bar{x}\bar{y} = b_1\left(\sum x_i^2 - n\bar{x}^2\right)$$

Therefore:

$$\boxed{\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2} = \frac{S_{xy}}{S_{xx}}}$$

where $S_{xy} = \sum(x_i - \bar{x})(y_i - \bar{y})$ and $S_{xx} = \sum(x_i - \bar{x})^2$.

And from equation (1):

$$\boxed{\hat{\beta}_0 = \bar{y} - \hat{\beta}_1\bar{x}}$$

### 15.3.3 Verifying the Minimum

The Hessian matrix of $S$ is:

$$H = 2\begin{pmatrix} n & \sum x_i \\ \sum x_i & \sum x_i^2 \end{pmatrix}$$

This is positive definite whenever $S_{xx} > 0$ (i.e., the $x$ values are not all identical), confirming that our solution is indeed a minimum.

### 15.3.4 An Important Identity: the Line Passes Through $(\bar{x}, \bar{y})$

From $\hat{\beta}_0 = \bar{y} - \hat{\beta}_1\bar{x}$, we see that:

$$\hat{y}(\bar{x}) = \hat{\beta}_0 + \hat{\beta}_1\bar{x} = \bar{y}$$

The fitted regression line always passes through the point of means $(\bar{x}, \bar{y})$.

## 15.4 Implementation from Scratch

Let us now implement OLS without using `lm()`. We build the estimators directly from the formulae we derived.


``` r
#' Ordinary Least Squares -- simple linear regression from scratch
#' @param x numeric vector of predictor values
#' @param y numeric vector of response values
#' @return named list with intercept, slope, fitted, residuals
my_ols <- function(x, y) {
  n <- length(x)
  stopifnot(length(y) == n, n >= 2)

  x_bar <- mean(x)
  y_bar <- mean(y)

  Sxy <- sum((x - x_bar) * (y - y_bar))
  Sxx <- sum((x - x_bar)^2)

  beta1 <- Sxy / Sxx
  beta0 <- y_bar - beta1 * x_bar

  fitted <- beta0 + beta1 * x
  residuals <- y - fitted

  list(
    intercept = beta0,
    slope = beta1,
    fitted = fitted,
    residuals = residuals,
    SSR = sum(residuals^2),
    n = n
  )
}
```


``` r
fit_scratch <- my_ols(cats$Bwt, cats$Hwt)

data.frame(
  parameter = c("intercept", "slope", "SSR"),
  from_scratch = c(fit_scratch$intercept, fit_scratch$slope, fit_scratch$SSR)
)
```

## 15.5 Understanding What OLS Does Geometrically

### 15.5.2 Properties of the Residuals

The OLS residuals satisfy two important identities. These follow directly from the normal equations.

**Property 1.** The residuals sum to zero:

$$\sum_{i=1}^{n} e_i = 0$$

*Proof*: from $\partial S/\partial b_0 = 0$ we obtained $\sum(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i) = 0$, which is $\sum e_i = 0$.

**Property 2.** The residuals are orthogonal to the predictor:

$$\sum_{i=1}^{n} x_i e_i = 0$$

*Proof*: from $\partial S/\partial b_1 = 0$ we obtained $\sum x_i(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i) = 0$, which is $\sum x_i e_i = 0$.


``` r
# Verify numerically

  "sum of residuals" = sum(fit_scratch$residuals),
  "sum of x*residuals" = sum(cats$Bwt * fit_scratch$residuals)
)
```

```

```

Both sums are zero (up to floating-point precision).

## 15.6 Decomposition of Total Variation

### 15.6.1 SST = SSR + SSE

Define:

- $\text{SST} = \sum(y_i - \bar{y})^2$ -- **Total Sum of Squares** (total variation in $Y$).
- $\text{SSReg} = \sum(\hat{y}_i - \bar{y})^2$ -- **Regression Sum of Squares** (variation explained by the model).
- $\text{SSE} = \sum(y_i - \hat{y}_i)^2 = \sum e_i^2$ -- **Error Sum of Squares** (unexplained variation).

**Theorem.** $\text{SST} = \text{SSReg} + \text{SSE}$.

*Proof*:

$$y_i - \bar{y} = (\hat{y}_i - \bar{y}) + (y_i - \hat{y}_i) = (\hat{y}_i - \bar{y}) + e_i$$

Squaring both sides and summing:

$$\text{SST} = \text{SSReg} + \text{SSE} + 2\sum(\hat{y}_i - \bar{y})e_i$$

The cross-term vanishes because $\hat{y}_i - \bar{y} = \hat{\beta}_1(x_i - \bar{x})$ and $\sum(x_i - \bar{x})e_i = \sum x_i e_i - \bar{x}\sum e_i = 0 - 0 = 0$. $\square$


``` r
SST <- sum((cats$Hwt - mean(cats$Hwt))^2)
SSReg <- sum((fit_scratch$fitted - mean(cats$Hwt))^2)
SSE <- fit_scratch$SSR

data.frame(
  quantity = c("SST", "SSReg", "SSE", "SSReg + SSE"),
  value = c(SST, SSReg, SSE, SSReg + SSE)
)
```

```
     quantity    value
1         SST 847.6256
2       SSReg 548.0924
3         SSE 299.5331
4 SSReg + SSE 847.6256
```

## 15.7 Estimating $\sigma^2$

The error variance $\sigma^2 = \text{Var}(\varepsilon_i)$ is unknown and must be estimated. The natural estimator is:

$$\hat{\sigma}^2 = s^2 = \frac{\text{SSE}}{n - 2} = \frac{\sum e_i^2}{n - 2}$$

We divide by $n - 2$ (not $n$) because we estimated two parameters ($\beta_0$ and $\beta_1$), consuming two degrees of freedom.

**Theorem.** $E[s^2] = \sigma^2$ -- the estimator is unbiased.


``` r
n <- nrow(cats)
s2 <- SSE / (n - 2)

c(
  "s^2 (from scratch)" = s2,
  "s^2 (from lm)" = summary(fit_lm)$sigma^2
)
```

```
s^2 (from scratch)      s^2 (from lm) 
          2.109388           2.109388 
```

The quantity $s = \sqrt{s^2}$ is called the **residual standard error** and is reported by R's `summary.lm()`.

## 15.8 Alternative Formula for $\hat{\beta}_1$

It is sometimes useful to write the slope estimator in terms of the sample correlation and sample standard deviations:

$$\hat{\beta}_1 = r_{xy} \cdot \frac{s_y}{s_x}$$

where $r_{xy}$ is the Pearson correlation coefficient. This shows that the sign of the slope is determined entirely by the sign of the correlation, and the magnitude is the correlation scaled by the ratio of standard deviations.


``` r
r_xy <- cor(cats$Bwt, cats$Hwt)
s_x <- sd(cats$Bwt)
s_y <- sd(cats$Hwt)

c(
  "beta1 from formula" = fit_scratch$slope,
  "r * sy/sx" = r_xy * s_y / s_x
)
```

```
beta1 from formula          r * sy/sx 
          4.034063           4.034063 
```

## Communicating to Stakeholders

When explaining a simple linear regression:

- **Avoid jargon**: instead of "the OLS slope estimate is 4.03", say "for every additional kilogram of body weight, we expect the heart to weigh about 4 grams more."
- **Always show the scatter plot with the fitted line**. The line alone is meaningless without seeing the data it summarises.
- **Report the intercept carefully**. If $X = 0$ is outside the range of the data (as it often is), do not interpret the intercept literally. Say "the intercept is a mathematical artefact needed to position the line correctly."
- **Mention the residual standard error** in practical units: "our predictions are typically off by about $\pm s$ units."
