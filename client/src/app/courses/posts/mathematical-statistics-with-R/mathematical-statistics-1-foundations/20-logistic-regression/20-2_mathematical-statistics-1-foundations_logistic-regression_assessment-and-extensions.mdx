# Chapter 20 -- Logistic Regression

## Part 2: Assessment and Extensions

In Part 1 we fitted logistic regression models and interpreted coefficients as odds ratios. But how well does the model *predict*? In this part we develop a crash course on classification metrics -- sensitivity, specificity, precision, recall, ROC curves, and AUC -- and then extend logistic regression to multinomial and ordinal outcomes.

---

## 20.6 Classification Metrics

### 20.6.1 The Confusion Matrix

When we use logistic regression for prediction, we convert the estimated probability $\hat{p}_i$ into a class label using a **threshold** $c$ (typically 0.5):

$$\hat{Y}_i = \begin{cases} 1 & \text{if } \hat{p}_i \ge c \\ 0 & \text{if } \hat{p}_i < c \end{cases}$$

The confusion matrix tabulates predictions against actual values:

|  | Predicted Positive | Predicted Negative |
|:---:|:---:|:---:|
| **Actual Positive** | True Positive (TP) | False Negative (FN) |
| **Actual Negative** | False Positive (FP) | True Negative (TN) |
