## 3.1 What Is a Random Variable?

In everyday language, a **random variable** is a rule that assigns a number to each possible outcome of a random experiment. If you roll a die and record the number showing, you have a random variable. If you flip three coins and count the heads, you have a random variable. The key idea is deceptively simple: *a random variable is not a number — it is a function*.

### Formal Definition

Let $(\Omega, \mathcal{F}, P)$ be a probability space. A **random variable** is a function

$$
X: \Omega \to \mathbb{R}
$$

that maps each outcome $\omega \in \Omega$ to a real number $X(\omega)$, with the additional requirement that for every real number $x$, the set $\{\omega \in \Omega : X(\omega) \leq x\}$ belongs to $\mathcal{F}$ (i.e., is a measurable set).

In plain English: $X$ takes something from the abstract world of random outcomes and converts it into a number we can do arithmetic with. The measurability condition simply ensures we can compute probabilities involving $X$.

### Why This Matters

The distinction between the random variable (the function) and any particular value it produces (a realisation) is fundamental. When we write $X$, we mean the entire mapping — all possible values at once, weighted by their probabilities. When we write $x$, we mean a specific number that $X$ might take. This notational convention pervades all of mathematical statistics.

### R Demonstration: Dice Rolling

Let us make this concrete. Consider rolling a fair die. The sample space is $\Omega = \{1, 2, 3, 4, 5, 6\}$, and the random variable $X(\omega) = \omega$ simply records the face value. In this case the mapping is trivial — but it need not be. We could define $Y(\omega) = \omega^2$ or $Z(\omega) = \mathbb{1}(\omega \text{ is even})$.

### Example: Counting Heads in Three Coin Flips

Define $X$ = number of heads in three fair coin flips. The sample space has $2^3 = 8$ outcomes:

| Outcome $\omega$ | $X(\omega)$ |
|:---|:---|
| TTT | 0 |
| HTT, THT, TTH | 1 |
| HHT, HTH, THH | 2 |
| HHH | 3 |

The random variable $X$ compresses eight outcomes into four possible values: $\{0, 1, 2, 3\}$.

## 3.2 Discrete Random Variables

A random variable $X$ is **discrete** if it can take on a finite or countably infinite number of distinct values $x_1, x_2, x_3, \ldots$. Examples include counts (number of defects), integers (dice rolls), and indicators (success/failure).

### The Probability Mass Function (PMF)

The **probability mass function** (PMF) of a discrete random variable $X$ is the function

$$
p(x) = P(X = x)
$$

that gives the probability of each possible value.

The PMF must satisfy two properties:

1. **Non-negativity**: $p(x) \geq 0$ for all $x$.
2. **Normalisation**: $\sum_{x} p(x) = 1$, where the sum is over all possible values of $X$.

In plain English: every value has a non-negative probability, and the total probability across all values is exactly one.

### Implementing the PMF from Scratch


``` r
#' Compute the PMF for a discrete random variable
#'
#' @param values Numeric vector of possible values
#' @param probs Numeric vector of corresponding probabilities
#' @param x The value(s) at which to evaluate the PMF
#' @return Numeric vector of probabilities
my_pmf <- function(values, probs, x) {
  # Validate inputs
  stopifnot(length(values) == length(probs))
  stopifnot(all(probs >= 0))
  stopifnot(abs(sum(probs) - 1) < 1e-10)

  # For each query point, find matching value and return its probability
  sapply(x, function(xi) {
    idx <- which(values == xi)
    if (length(idx) == 0) return(0)
    probs[idx]
  })
}

# PMF for a fair die
die_values <- 1:6
die_probs <- rep(1/6, 6)

# PMF of a fair die
sapply(1:6, function(v) my_pmf(die_values, die_probs, v))

# PMF for number of heads in 3 coin flips
heads_values <- 0:3
heads_probs <- c(1/8, 3/8, 3/8, 1/8)

sapply(0:3, function(v) my_pmf(heads_values, heads_probs, v))

# Query a value not in the support -- returns 0
my_pmf(heads_values, heads_probs, 5)
```

### The Cumulative Distribution Function (CDF)

The **cumulative distribution function** (CDF) of any random variable $X$ is

$$
F(x) = P(X \leq x)
$$

For a discrete random variable, this becomes

$$
F(x) = \sum_{t \leq x} p(t)
$$

The CDF accumulates probability as we move from left to right along the number line. It has the following properties:

1. $F$ is non-decreasing: if $a < b$ then $F(a) \leq F(b)$.
2. $\lim_{x \to -\infty} F(x) = 0$ and $\lim_{x \to \infty} F(x) = 1$.
3. $F$ is right-continuous: $\lim_{h \to 0^+} F(x + h) = F(x)$.

For discrete random variables, the CDF is a **step function** — it jumps at each value in the support and is flat in between.

### Implementing the Discrete CDF from Scratch


``` r
#' Compute the CDF for a discrete random variable
#'
#' @param values Numeric vector of possible values (sorted)
#' @param probs Numeric vector of corresponding probabilities
#' @param x The value(s) at which to evaluate the CDF
#' @return Numeric vector of cumulative probabilities
my_cdf_discrete <- function(values, probs, x) {
  # Ensure values are sorted
  ord <- order(values)
  values <- values[ord]
  probs <- probs[ord]

  sapply(x, function(xi) {
    sum(probs[values <= xi])
  })
}

# CDF of a fair die
sapply(0:7, function(v) my_cdf_discrete(die_values, die_probs, v))
```

## 3.3 Continuous Random Variables

A random variable $X$ is **continuous** if it can take on any value in an interval (or union of intervals) of the real line. Examples include Height, Weight, temperature, and time.

Before we can properly define continuous random variables, we need a brief detour into integration.

### Crash Course: Integration Basics

**What is an integral?** The definite integral $\int_a^b f(x)\,dx$ represents the **signed area** under the curve $y = f(x)$ between $x = a$ and $x = b$. If $f(x) \geq 0$ on $[a, b]$, the integral is simply the area.

Geometrically, imagine approximating the area by stacking thin rectangles of width $\Delta x$ and height $f(x_i)$. As the rectangles get thinner ($\Delta x \to 0$), the sum of their areas converges to the integral:

$$
\int_a^b f(x)\,dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i) \Delta x
$$

**Key properties** of integrals:

1. **Linearity**: $\int_a^b [c_1 f(x) + c_2 g(x)]\,dx = c_1\int_a^b f(x)\,dx + c_2\int_a^b g(x)\,dx$
2. **Additivity over intervals**: $\int_a^c f(x)\,dx = \int_a^b f(x)\,dx + \int_b^c f(x)\,dx$
3. **Fundamental Theorem of Calculus**: If $F'(x) = f(x)$, then $\int_a^b f(x)\,dx = F(b) - F(a)$

This is not a calculus course. You need only enough intuition to read the definitions that follow: an integral accumulates a function's values over an interval, just as a sum accumulates discrete values.


``` r
# R's integrate() function computes definite integrals numerically

# integral of x^2 from 0 to 1 (should be 1/3)
integrate(function(x) x^2, lower = 0, upper = 1)

# integral of exp(-x) from 0 to infinity (should be 1)
integrate(function(x) exp(-x), lower = 0, upper = Inf)

# integral of dnorm(x) from -Inf to Inf (should be 1)
integrate(dnorm, lower = -Inf, upper = Inf)
```

### The Probability Density Function (PDF)

A random variable $X$ is continuous if there exists a non-negative function $f(x)$, called the **probability density function** (PDF), such that for any interval $[a, b]$:

$$
P(a \leq X \leq b) = \int_a^b f(x)\,dx
$$

The PDF must satisfy:

1. **Non-negativity**: $f(x) \geq 0$ for all $x$.
2. **Normalisation**: $\int_{-\infty}^{\infty} f(x)\,dx = 1$.

### A Critical Insight: Density Is Not Probability

This is perhaps the single most confusing point for newcomers to continuous probability, so let us be very clear:

**$f(x)$ is NOT a probability.** It is a *density*. For a continuous random variable, $P(X = x) = 0$ for every single value $x$.

Why? Because $P(X = x) = P(x \leq X \leq x) = \int_x^x f(t)\,dt = 0$. The integral over an interval of zero width is zero.

This means that for continuous random variables, asking "what is the probability that $X$ equals exactly 3.7?" is meaningless — the answer is always zero. Instead, we ask "what is the probability that $X$ falls between 3.5 and 3.9?"

**Analogy**: Think of a thin metal rod with varying density (mass per unit length). The density at a point tells you how concentrated the mass is *near* that point, but a single point has zero length and therefore zero mass. To find the mass of a section, you must integrate the density over an interval. Probability density works exactly the same way.

Note that $f(x)$ can exceed 1. For example, the uniform distribution on $[0, 0.5]$ has $f(x) = 2$ for $x \in [0, 0.5]$. This is perfectly fine — densities are not probabilities.

### CDF for Continuous Random Variables

The CDF of a continuous random variable is

$$
F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)\,dt
$$

and the fundamental relationship between PDF and CDF is:

$$
f(x) = F'(x) = \frac{d}{dx}F(x)
$$

The PDF is the derivative of the CDF. Equivalently, the CDF is the integral (antiderivative) of the PDF. This is the continuous analogue of the discrete relationship $p(x) = F(x) - F(x^-)$, where $F(x^-)$ is the left-hand limit.

### R Demonstration: The Uniform Distribution

The simplest continuous distribution is the **uniform distribution** on $[a, b]$:

$$
f(x) = \begin{cases} \frac{1}{b - a} & \text{if } a \leq x \leq b \\ 0 & \text{otherwise} \end{cases}
$$

Every value in $[a, b]$ is equally likely (in the density sense). Let us implement this and verify it.


``` r
# Uniform(0, 1) PDF and CDF from scratch
my_uniform_pdf <- function(x, a = 0, b = 1) {
  ifelse(x >= a & x <= b, 1 / (b - a), 0)
}

my_uniform_cdf <- function(x, a = 0, b = 1) {
  ifelse(x < a, 0, ifelse(x > b, 1, (x - a) / (b - a)))
}

# Verify normalisation (should be 1)
integrate(my_uniform_pdf, lower = -Inf, upper = Inf)

# Compute P(0.3 <= X <= 0.7) (exact: 0.4)
integrate(my_uniform_pdf, lower = 0.3, upper = 0.7)

# Equivalently via CDF
my_uniform_cdf(0.7) - my_uniform_cdf(0.3)
```

```
1 with absolute error < 4.2e-11
0.4 with absolute error < 4.4e-15
[1] 0.4
```

## 3.4 The Quantile Function

The **quantile function** (or inverse CDF) is defined as:

$$
Q(p) = F^{-1}(p) = \inf\{x : F(x) \geq p\} \quad \text{for } 0 < p < 1
$$

In plain English: $Q(p)$ is the smallest value $x$ such that the probability of $X$ being at most $x$ is at least $p$. The quantile function answers the question: "Below what value does $p \times 100\%$ of the distribution fall?"

For continuous distributions with strictly increasing CDFs, the quantile function is simply the algebraic inverse of $F$. For discrete distributions, we use the infimum definition because the CDF has flat regions.

### The Quantile Function in R

R provides quantile functions for every built-in distribution, named with the `q` prefix:


``` r
# Standard normal: what value has 95% of the distribution below it?
q95 <- qnorm(0.95)
q95
pnorm(q95)  # verify: should be 0.95

# Median of an Exponential(rate = 2)
med <- qexp(0.5, rate = 2)
med
pexp(med, rate = 2)  # verify: should be 0.5

# Quartiles of a Chi-squared(df = 5)
qchisq(c(0.25, 0.50, 0.75), df = 5)
```

The CDF and quantile function are mirror images of each other, reflected across the line $y = x$. The CDF asks "given a value, what probability?"; the quantile function asks "given a probability, what value?"

### The Four Functions in R

For every distribution in R, there are four functions following a consistent naming convention:

| Prefix | Function | Example (Normal) |
|:---|:---|:---|
| `d` | Density/mass function | `dnorm(x)` |
| `p` | CDF (cumulative probability) | `pnorm(q)` |
| `q` | Quantile function (inverse CDF) | `qnorm(p)` |
| `r` | Random number generation | `rnorm(n)` |

This pattern holds for `binom`, `pois`, `unif`, `exp`, `gamma`, `chisq`, `t`, `f`, and many more.


``` r
# The four normal distribution functions
dnorm(0)       # density at x = 0
pnorm(0)       # CDF at x = 0
qnorm(0.975)   # 97.5th percentile

set.seed(42)
rnorm(5)        # 5 random draws
```

## Communicating to Stakeholders

When explaining random variables to a non-technical audience, focus on the mapping idea: "A random variable is a rule that converts uncertain outcomes into numbers." Avoid the formal measure-theoretic definition. Instead, use concrete examples: "If we randomly sample a customer, their spending amount is a random variable."

For the PMF/PDF distinction, use the weight analogy: in a bag of distinctly labelled balls, each ball has a specific probability of being drawn (PMF). In contrast, for something like Height, we talk about the probability of falling within a range, not of being exactly a particular value (PDF).

When discussing CDFs, the phrase "the probability of being at or below a threshold" is universally understood.

## Quick Reference

| Concept | Discrete | Continuous |
|:---|:---|:---|
| **Probability function** | PMF: $p(x) = P(X = x)$ | PDF: $f(x)$, where $P(a \leq X \leq b) = \int_a^b f(x)\,dx$ |
| **CDF** | $F(x) = \sum_{t \leq x} p(t)$ | $F(x) = \int_{-\infty}^{x} f(t)\,dt$ |
| **Key constraint** | $\sum_x p(x) = 1$ | $\int_{-\infty}^{\infty} f(x)\,dx = 1$ |
| **$P(X = x)$** | $= p(x) \geq 0$ | $= 0$ always |
| **Quantile** | $Q(p) = \inf\{x : F(x) \geq p\}$ | $Q(p) = F^{-1}(p)$ |
| **R density** | `dbinom(x, n, p)` | `dnorm(x, mu, sigma)` |
| **R CDF** | `pbinom(q, n, p)` | `pnorm(q, mu, sigma)` |
| **R quantile** | `qbinom(p, n, p)` | `qnorm(p, mu, sigma)` |
| **R random** | `rbinom(n, size, p)` | `rnorm(n, mu, sigma)` |
