<a id="tetracycline-responsive-element-tre-ptre">Tetracycline-Responsive Element (TRE/pTRE)</a> - A [promoter](#promoter) that contains operator sequences ([TetO](#teto)) for a [TetR](#tetr) [protein](#protein) to bind. The TRE [promoter](#promoter) contains multiple [TetO](#teto) sites and a minimal [CMV](#cytomegalovirus-cmv) core [promoter](#promoter).
