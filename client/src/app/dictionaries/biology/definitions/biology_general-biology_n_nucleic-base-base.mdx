<a id="nucleic-base-base">Nucleic Base (Base)</a> - In biology this refers to the basic building blocks of [DNA](#dna-desoxyribonucleic-acid) and [RNA](#rna). The primary nucleobases (nucleic bases) are **Cytosine (C), Guanine (G), Adenine (A), Thymine (T) and Uracil (U)**.
