<a id="codon">Codon</a> - A triplet (three) of [ribonucleotides](#ribonucleotides) in [mRNA](#mrna) that encodes for an [amino acid](#amino-acid).

 Note that whether or not a codon matches this table depends on the organism. Some organisms, especially [microorganisms](#micro-organism) and [bacteria](#bacteria) often present modified readings of these codons (known as alternative codons). This is dependant on the presence or not of a [tRNA](#trna) capable of matching the [DNA](#dna-desoxyribonucleic-acid) sequence (codon).

| CUG                      | CTG | Start    | Leucine    | Mammals Rare |

| AUU                      | ATT | Start    | Methionine |              |

| UUG                      | TTG | Start    | Methionine |              |
| UAA                      | TAA | Stop     | Ochre      | Universal    |
| UAG                      | TAG | Stop     | Amber      |              |
| UGA                      | TGA | Stop     | Opal       |              |
