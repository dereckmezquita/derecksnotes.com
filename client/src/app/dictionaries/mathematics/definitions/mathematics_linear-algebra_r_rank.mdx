<a id="rank">Rank</a> - In **linear algebra**, the **rank** of a [matrix](#matrix) $A$ is the dimension of its column space (or row space). Equivalently, itâ€™s the maximum number of linearly independent columns (or rows). The rank also ties into the concept of [dimension](#dimension) of the image (column space) of a linear transformation.

**Key points**:
- If $\mathrm{rank}(A) < \min(m,n)$ for an $m \times n$ matrix, the matrix is not full rank.
- The rank-nullity theorem: $\mathrm{rank}(A)+ \mathrm{nullity}(A) = n$.

**R demonstration** (finding rank of a matrix):


``` r
library(MASS)
library(data.table)

A <- matrix(c(1,2,3, 2,4,6, 3,6,9), nrow=3, byrow=TRUE)
A_rank <- qr(A)  # rank via QR decomposition
A_rank
```
