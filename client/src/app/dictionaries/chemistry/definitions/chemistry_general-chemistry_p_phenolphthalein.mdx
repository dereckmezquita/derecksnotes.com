<a id="phenolphthalein">Phenolphthalein</a> - Common [indicator](#indicator) in [acid_base](#acid-base) [titration](#titration)s. Changes from colourless to pink.

Properties:
- [pH](#p-h) range: 8.2-10.0
- Colourless pH < 8.2
- Pink pH > 10.0
- Sharp endpoint
- Formula: C₂₀H₁₄O₄

Best for:
- Strong acid/strong base titrations
- Weak acid/strong base titrations
- Carbonate determinations
