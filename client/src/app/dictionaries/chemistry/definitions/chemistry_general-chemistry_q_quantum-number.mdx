<a id="quantum_number">Quantum Number</a> - Four numbers describing an electron's quantum state in an [electron_shell](#electron_shell). Derived from [wave_function](#wave_function) solutions.

Principal (n):
- Energy level
- n = 1, 2, 3...
- Size of [orbital](#orbital)

Angular (l):
- Subshell shape
- l = 0 to (n-1)
- s(0), p(1), d(2), f(3)

Magnetic (mₗ):
- Orbital orientation
- mₗ = -l to +l
- p: 3 orientations

Spin (mₛ):
- Electron spin
- mₛ = ±½
