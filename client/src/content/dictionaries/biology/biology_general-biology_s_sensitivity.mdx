<a id="sensitivity">Sensitivity</a> - How well a method can pick up true positives; calculated by dividing the number of true positives by the number of true positives added to the number of false negatives.

$$
    Sensitivity = \frac{(True Positives)}{(True Positives) + (False Negatives)}
$$
